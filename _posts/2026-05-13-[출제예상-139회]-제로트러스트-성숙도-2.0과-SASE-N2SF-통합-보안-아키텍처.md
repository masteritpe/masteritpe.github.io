## 2. 개념·정의

### 제로트러스트(Zero Trust)
> "Never Trust, Always Verify" — 모든 사용자·디바이스·트래픽을 신뢰하지 않고, **지속적 인증·인가·검증**을 수행하는 보안 모델.

### 제로트러스트 성숙도 모델 2.0 (CISA, 한국 가이드라인 2.0)
- **5개 기둥(Pillars)**: 사용자·디바이스·네트워크·애플리케이션·데이터
- **4단계 성숙도**: Traditional → Initial → Advanced → Optimal
- **3개 횡단 기능**: 가시성·자동화·거버넌스

### SASE (Secure Access Service Edge)
> 네트워크(SD-WAN) + 보안(SWG·CASB·ZTNA·FWaaS) 기능을 **클라우드 엣지에서 통합 제공**하는 서비스 모델.

### N2SF (National Network Security Framework)
> 국가정보원이 공공기관 대상으로 발표(2025.01)한 **국가 망 보안체계** — 제로트러스트 원칙을 망분리·계정·접근통제에 적용.

## 4. 핵심 기술요소

### 4.1 5개 기둥 상세

| 기둥 | 핵심 통제 | 대표 기술 |
|------|----------|-----------|
| 사용자 | 강한 인증·지속 검증 | FIDO2, MFA, 위험 기반 인증, ITDR |
| 디바이스 | 무결성·상태 검증 | MDM, EDR, Hardware Attestation, TPM |
| 네트워크 | 마이크로 세그멘테이션 | SDP, ZTNA, Micro-segmentation, DNS Security |
| 애플리케이션 | 런타임 보호·API 통제 | RASP, WAAP, API Gateway, OAuth 2.1 |
| 데이터 | 분류·암호화·이동통제 | DLP, DRM, KMS, 마스킹, 토큰화 |

### 4.2 성숙도 4단계 (CISA 2.0)

```
Traditional  →  Initial   →   Advanced  →  Optimal
정적 정책        부분 자동화     정책 통합     완전 자동·동적
경계 신뢰        ID 중심         컨텍스트 기반  AI/ML 위협 대응
```

### 4.3 SASE 구성요소

- **SWG** (Secure Web Gateway): 웹 트래픽 검사·URL 필터링
- **CASB** (Cloud Access Security Broker): SaaS 가시성·통제
- **ZTNA** (Zero Trust Network Access): SDP 기반 애플리케이션 단위 접근
- **FWaaS**: 클라우드 방화벽
- **SD-WAN**: 광역 네트워크 최적화

### 4.4 N2SF 핵심 요구사항

- **단일 사용자 식별**: 통합 계정·MFA 의무화
- **장비 등록·인증**: 미등록 장비 차단
- **네트워크 마이크로 세그멘테이션**: 망 내 횡적 이동 제한
- **로깅·감사**: 모든 접근 로그 중앙 수집
- **공공·민간 정보 분리 통제**

## 5. 적용 고려사항

### 5.1 기존 망분리와의 관계
- **기존 망분리(에어갭)**: 물리·논리 분리, 변경 어려움
- **제로트러스트 전환**: 망분리 유지 + 망 내부에서 마이크로 세그멘테이션 추가
- **하이브리드 전략**: 민감 자산은 망분리 유지, 일반 업무는 ZTNA로 점진 전환

### 5.2 도입 단계
1. **자산·트래픽 가시화** (CMDB·NDR)
2. **사용자·디바이스 ID 통합** (IAM·MDM)
3. **마이크로 세그멘테이션 시범 적용**
4. **ZTNA·SASE 단계적 확대**
5. **정책 자동화·SOAR 연계**

### 5.3 주요 도전 과제

| 과제 | 대응 |
|------|------|
| 레거시 시스템 비호환 | API 프록시·Identity-aware Proxy |
| 가시성 부족 | Network TAP·NDR·UEBA |
| 정책 폭증 | 정책 엔진 통합·자동화(OPA) |
| 사용자 경험 저하 | SSO·SDP·Passive Verification |
| 클라우드·온프레미스 혼재 | SASE 단일 콘솔 |
