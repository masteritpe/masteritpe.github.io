## 1. 뉴스 요약

동아사이언스는 국내 연구진이 기존 메모리 대비 동작 전력을 66분의 1 수준으로 낮춘 온도 기반 데이터 저장 메모리 기술을 개발했다고 보도했다. 발표된 기술은 전기 펄스가 아닌 국소 가열에 의한 결정 상태(상전이) 변화를 활용해 비휘발성으로 데이터를 보존하며, 셀당 발열·소모 전력을 크게 줄여 AI 데이터센터의 냉각·전력 비용 문제 해소에 기여할 전망이다. 이는 D램·플래시 양극화 구조를 보완하는 차세대 비휘발성 메모리(NVM) 카테고리의 한 갈래로, 메모리 계층 구조에서 'Storage Class Memory(SCM)' 영역을 차지하는 신기술 흐름과 맞닿아 있다. 글로벌 차원에서 인텔(Optane 단종 이후), SK하이닉스·삼성전자, IMEC 등이 PCM·MRAM·ReRAM·FeRAM 계열 양산 기술을 경쟁 중인 상황에서 국내 학계 성과의 의미가 크다.

## 2. 핵심 개념

### 메모리 계층(Memory Hierarchy)과 SCM 위치

```
┌─────────────────────────────────────────────────────────────┐
│  계층     매체 예시          지연      용량밀도   휘발성       │
├─────────────────────────────────────────────────────────────┤
│  L1/L2   SRAM              ~1ns      낮음       휘발          │
│  Main    DRAM/HBM          ~80ns     중간       휘발          │
│ ★SCM ★  PCM/MRAM/ReRAM     100ns~1μs 높음       비휘발 ←신기술│
│  SSD     NAND Flash        ~50μs     매우높음    비휘발        │
│  HDD     자기 디스크        ~5ms      최고       비휘발        │
└─────────────────────────────────────────────────────────────┘
        ↑
   온도 기반 PCM은 SRAM/DRAM과 SSD 사이의 'Persistent Memory' 영역을 채움
```

### 차세대 비휘발성 메모리 비교

아래는 주요 NVM 후보 기술의 동작 원리·특성 비교다.

| 기술 | 저장 원리 | Endurance | 지연 | 전력 |
|------|----------|-----------|------|------|
| PCM(상변화) | Crystalline↔Amorphous 상전이 | 10^8~10^9 | ~100ns | 높음(기존)→저전력(신기술) |
| MRAM(자기) | 스핀 토크 자화 방향 | 10^15 | 10ns | 매우 낮음 |
| ReRAM(저항) | 저항성 산화막 필라멘트 | 10^6~10^9 | ~10ns | 낮음 |
| FeRAM(강유전) | 분극 방향 | 10^14 | ~50ns | 매우 낮음 |
| NAND(전하) | 플로팅 게이트 전자 | 10^4~10^5 | ~50μs | 중간 |

### 온도 기반(저전력 PCM) 동작 원리

- **셀 구조**: GST(Ge2Sb2Te5) 등 칼코게나이드 박막 + 국소 히터 전극
- **Set/Reset**: 짧은 고전류 펄스(Reset, 비정질) ↔ 긴 저전류 펄스(Set, 결정) → 저항 차로 0/1 식별
- **저전력화 메커니즘**: 발열 면적을 nm급으로 미세화하고 상변화 임계 온도를 낮춤 → 동일 상전이에 필요 줄(J)을 1/66로 절감
- **다단 셀(MLC)**: 중간 결정도를 활용한 2bit/cell, 밀도 향상 가능

### 시스템 통합 구조

```
[CPU 코어] ─┬─ HBM/DRAM   (휘발, 작업 메모리)
            │
            ├─ CXL.mem   (메모리 풀링·확장)
            │
            └─ SCM(PCM)  (영속·저전력, 페이지 캐시·KV 스토어 가속)
                  │
                  └─ NVMe SSD (대용량 영속)
```

CXL(Compute Express Link) 메모리 코히어런시 표준을 통해 CPU·GPU·가속기가 SCM을 풀(pool)로 공유, 메모리 확장과 데이터 이동 비용을 동시에 줄이는 구조가 부상하고 있다.

## 3. 기술사 출제 포인트

### 논술형 예상 주제

- 차세대 비휘발성 메모리(NVM) 기술 동향 및 메모리 계층 구조 변화 방안
- AI 데이터센터의 전력·냉각 문제와 저전력 메모리·CXL 기반 메모리 풀링 전략
- Storage Class Memory(SCM)의 시스템 아키텍처 통합 방안과 OS·DBMS 영향

### 단답형 핵심 키워드

- **PCM(Phase-Change Memory)**: 칼코게나이드 박막의 상전이를 이용한 비휘발성 메모리
- **SCM(Storage Class Memory)**: DRAM과 SSD 사이 지연·내구성을 채우는 영속 메모리 계층
- **CXL(Compute Express Link)**: PCIe 기반 캐시 코히어런트 인터커넥트, 메모리 풀링 지원(CXL.mem)
- **Endurance**: 메모리 셀이 견딜 수 있는 쓰기 횟수, NVM의 핵심 지표
- **Wear Leveling**: 쓰기 횟수가 한정된 NVM에서 셀 마모를 균등화하는 컨트롤러 알고리즘
- **Persistent Memory Programming**: PMDK·DAX 등 OS·라이브러리 차원의 영속 메모리 프로그래밍 모델

### 비교·대조 포인트

아래는 DRAM 중심 아키텍처와 SCM 통합 아키텍처 비교다.

| 구분 | DRAM 중심 | SCM 통합 아키텍처 |
|------|-----------|------------------|
| 데이터 영속성 | 휘발(전원 차단 시 소실) | 비휘발(즉시 영속) |
| 시동 시간 | 콜드부트·로딩 필수 | Instant-On 가능 |
| 전력(idle) | 높음(refresh 필요) | 매우 낮음(refresh 불필요) |
| DBMS 영향 | WAL·체크포인트 의존 | Log-less·In-place update 가능 |
| 메모리 확장 | DIMM 슬롯 한계 | CXL 풀링으로 TB급 확장 |
| 비용/GB | 높음 | 중간(NAND>SCM>DRAM) |
