## 1. 뉴스 요약

오라클이 AI 데이터베이스 업데이트를 통해 핵심 업무에 에이전틱 AI를 지원하겠다고 발표했다. 에이전틱 AI가 데이터베이스 운영·최적화·보안에 직접 개입하는 '자율운영 DB(Autonomous Database)' 개념이 확산되고 있다. 기존 DBA 중심의 수동 관리에서 AI 에이전트가 쿼리 최적화, 인덱스 관리, 장애 예측, 자동 스케일링까지 수행하는 패러다임 전환이 진행 중이다. 국내에서도 제조·금융·공공 분야에서 AI 에이전트 기반 데이터 관리 도입이 본격화되고 있다.

## 2. 핵심 개념

### 자율운영 DB 발전 단계

| 단계 | 자동화 수준 | DBA 역할 | 핵심 기능 |
|------|-------------|----------|-----------|
| Level 1: 수동 관리 | 없음 | 전적 운영 | 수동 튜닝, 백업, 패치 |
| Level 2: 자동화 보조 | 부분 자동화 | 의사결정 중심 | 자동 백업, 알림 기반 모니터링 |
| Level 3: 자율 튜닝 | ML 기반 최적화 | 감독·승인 | 셀프 튜닝, 자동 인덱싱, 이상 탐지 |
| Level 4: 자율 운영 | 에이전틱 AI | 전략·거버넌스 | 자율 장애복구, 예측 스케일링, 보안 자동화 |
| Level 5: 완전 자율 | 범용 AI | 최소 개입 | 비즈니스 목표 기반 전체 자율 관리 |

### 에이전틱 AI 데이터베이스 핵심 기능

| 기능 영역 | 전통 방식 | 에이전틱 AI 방식 |
|-----------|-----------|------------------|
| 쿼리 최적화 | DBA 수동 튜닝, 실행계획 분석 | AI가 워크로드 패턴 학습 후 자동 최적화 |
| 인덱스 관리 | 주기적 분석 후 수동 생성/삭제 | 워크로드 변화에 따른 자동 인덱스 추천·생성 |
| 용량 관리 | 수동 스케일업/아웃 | 부하 예측 기반 선제적 자동 스케일링 |
| 장애 대응 | 알림 후 수동 조치 | 이상 징후 사전 감지 → 자동 복구 실행 |
| 보안 관리 | 정기 감사, 수동 패치 | 실시간 이상 접근 탐지, 자동 패치 적용 |
| 성능 분석 | AWR/ASH 리포트 수동 분석 | 자연어 질의로 성능 이슈 진단 및 해결 |

### 에이전틱 AI DB의 기술 구성

- **관찰(Observe)**: 실시간 메트릭 수집 (CPU, I/O, 대기 이벤트, 쿼리 로그)
- **분석(Analyze)**: ML 모델로 워크로드 패턴 분류, 이상 탐지, 근본 원인 분석
- **계획(Plan)**: 최적 조치 방안 수립 (인덱스 생성, 파라미터 변경, 스케일링)
- **실행(Act)**: 승인 정책에 따라 자동 실행 또는 DBA 승인 요청
- **학습(Learn)**: 조치 결과 피드백 반영, 모델 지속 개선

## 3. 기술사 출제 포인트

### 예상 문제
1. **자율운영 데이터베이스(Autonomous Database)의 개념과 발전 단계를 설명하시오**
2. **에이전틱 AI를 활용한 데이터베이스 자동 튜닝 기술의 구현 방식과 기존 DBA 운영 모델과의 차이점을 비교하시오**
3. **AI 기반 데이터베이스 이상 탐지 및 자동 복구 시스템의 아키텍처를 설계하시오**

### 답안 구성 가이드
- 자율운영 DB의 5단계 성숙도 모델과 각 단계별 핵심 기술 요소
- 에이전틱 AI의 Observe-Analyze-Plan-Act-Learn 루프 구조
- 셀프 드라이빙(Self-Driving), 셀프 시큐어링(Self-Securing), 셀프 리페어링(Self-Repairing) 3대 축
- DBA 역할 변화: 운영 관리자 → 데이터 전략가/거버넌스 관리자
- 실무 고려사항: AI 판단의 투명성(Explainability), 자동화 범위 제한(Guardrail), 감사 추적(Audit Trail)

## 4. 관련 토픽 연계

- **AIOps**: IT 운영 전반의 AI 자동화와 DB 자율운영의 관계
- **옵저버빌리티(Observability)**: 메트릭·로그·트레이스 통합 모니터링 체계
- **데이터베이스 튜닝**: 쿼리 최적화, 실행계획 분석의 AI 자동화 방향
- **클라우드 네이티브 DB**: 서버리스 DB와 자동 스케일링 아키텍처
- **에이전틱 AI**: 자율적 목표 수행 AI 에이전트의 아키텍처와 오케스트레이션
