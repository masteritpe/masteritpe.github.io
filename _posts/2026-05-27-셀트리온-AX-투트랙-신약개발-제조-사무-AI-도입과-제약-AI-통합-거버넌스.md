## 1. 뉴스 요약

2026-05-25~26 헤럴드경제·네이트·조선일보·KBS 등 다수 매체가 "**셀트리온, 신약개발·제조·사무 전반 AI 확대…AX 투트랙 가동**"을 보도했다. 핵심은 ① 신약 R&D(타깃 발굴·후보 물질 설계), ② 생산(공정 모니터링·품질 예측·예지보전), ③ 사무·운영(문서·법무·인사·재무) 전 영역에 AI를 동시 도입하는 **AX 투트랙 전략**이며 ② **생명과학 도메인 LLM·RAG·MLOps** 와 **GMP/GxP·규제 준수 거버넌스**를 결합한 산업 AX 모델이라는 점이다. 동일 시기 BC카드 AI 상담센터, KB금융 AI 보안 고도화, K-AI 정부 도입 보도가 함께 나오며 산업 전반의 전사 AX 흐름이 가속화되고 있다. 앞서 **앤트로픽-PwC 클로드 엔터프라이즈 AI 대전환(2026-05-18)**, **AX 기업 AI전환과 소프트웨어 개발 패러다임 변화(2026-03-23)** 흐름과 연결되며, **제약·바이오 특수 규제** 영역의 AI 도입 패턴이라는 차별성을 가진다. 기술사 소프트웨어공학 영역에서 **기업 AX·AI 도입 거버넌스·MLOps**는 출제 가능성이 큰 토픽이다.

## 2. 핵심 개념

### 셀트리온 AX 투트랙 개요

| 항목 | 내용 |
|---|---|
| 주관 | 셀트리온(바이오시밀러·신약·제조) |
| 시점 | 2026-05 발표, 전사 AI 도입 가속화 |
| 트랙 1 | 신약개발·제조 핵심 가치사슬(R&D+Manufacturing) |
| 트랙 2 | 사무·운영 백오피스(법무·인사·재무·구매) |
| 도입 패턴 | 도메인 특화 LLM + RAG + Agent + MLOps |
| 규제 맥락 | GMP, GxP, ICH, FDA/EMA/MFDS, ALCOA+ 데이터 무결성 |

### AX 도입 영역별 AI 적용

| 영역 | 주요 AI 활용 | 핵심 데이터 |
|---|---|---|
| 신약 R&D | 단백질 구조 예측(AlphaFold), 결합 후보 생성, 임상 데이터 분석 | 단백질 서열, ADMET, 임상 RWD |
| 제조·품질 | 공정 이상탐지, 품질 예측, 예지보전, MES 연계 자율운영 | 센서 시계열, 배치 레코드, LIMS |
| 임상·약물감시 | 환자 코호트 분석, 시험 설계, 부작용 자동 분류 | EHR, EDC, 약물감시 보고 |
| 사무·운영 | 계약·법무 RAG, HR 봇, 재무 분석, 구매 협상 보조 | 사내 문서, ERP, 정책 가이드 |
| 영업·마케팅 | 의사·약사 상담 어시스턴트, 시장 분석 | CRM, 의학정보 콘텐츠 |

### 일반 기업 AX vs 제약·바이오 AX

| 구분 | 일반 기업 AX | 제약·바이오 AX |
|---|---|---|
| 규제 강도 | 산업·국가 일반 규제 | GMP/GxP/ICH, 제품 인허가 |
| 데이터 무결성 | 일반 데이터 품질 | ALCOA+ 원칙(가용·읽기·동시·원본·정확) |
| 검증 절차 | 비즈니스 검증 | CSV/CSA 컴퓨터시스템 검증, 변경관리 |
| 책임 | 사업 책임자 | 제조소·품질·약사 책임 명확화 |
| 학습 데이터 | 운영·고객 데이터 | 임상·환자정보(가명·익명·동의 관리) |
| 모델 변경 | 신속 배포 | 변경관리 절차·재검증 필요 |

## 3. 기술사 출제 포인트

### 개념 정의와 필요성

- **AX(AI Transformation)**: 기존 DX(디지털 전환)에서 한 단계 나아가 AI를 핵심 운영 동력으로 내재화하는 기업 변혁
- **AX 투트랙**: 핵심 가치사슬(R&D·제조)과 운영(사무·관리) 영역을 동시에 변환해 시너지를 극대화하는 전략
- **필요성**: ① 신약 개발 비용·기간 감축, ② 글로벌 인허가 대응 속도, ③ 제조 품질·안전성, ④ 인력·자원 효율화, ⑤ 글로벌 경쟁사 대비 속도 우위

### 기존 방식과의 비교

- **부서별 파일럿**: 점단위 PoC 누적, 데이터·플랫폼 분절. ROI 입증 어려움
- **외주 SaaS 도입**: 빠르지만 도메인 적합성·데이터 통제 한계
- **단일 LLM API 적용**: 의약·임상 도메인 환각 위험, 규제 검증 미흡
- **전사 AX 플랫폼(셀트리온형)**: 사내 LLM·RAG·MLOps·거버넌스 통합, 도메인 특화 + 규제 적합

### 아키텍처 구성요소

- **AI 플랫폼**: 사내 GPU·KGB(Knowledge Graph Base), MLOps, LLMOps, 벡터 DB
- **도메인 LLM**: 의학·약학·제조 코퍼스 미세조정, 도메인 RAG(검색 증강 생성)
- **데이터 패브릭**: 임상·실험·제조·운영 데이터 통합 카탈로그, 가명·익명·동의 메타데이터
- **에이전트 오케스트레이션**: 업무 에이전트(계약·연구·품질) + MCP/Function Calling
- **MLOps + ClinOps**: 모델 등록·검증·승인·배포·모니터링, CSV/CSA 절차 통합
- **거버넌스 보드**: AI 윤리·위험 평가, 모델 카드·데이터시트, ISO/IEC 42001 AIMS 적용
- **보안**: 사내망/하이브리드 클라우드, PII·임상정보 토큰화, 감사 로그 보존
- **인적 자원**: 시민 개발자·도메인 전문가-MLE 협업, AX 리스킬링 프로그램

### 도입 시 고려사항

- **규제 적합성**: FDA AI/ML 가이드, EMA·MFDS 가이드라인 모니터링, ICH E6(R3) 임상 GCP
- **데이터 무결성(ALCOA+)**: 원본성·동시성·정확성 보장 설계, 변경 로그·감사 추적
- **검증·변경관리**: 모델 변경·미세조정마다 CSV/CSA 재검증, 임팩트 평가
- **개인정보·민감정보**: 의료법·개인정보보호법·GDPR 동시 준수, 가명정보 활용 특례 적용
- **윤리·환각**: 의사·약사 임상 의사결정 보조에 한정(HITL), 권한 분리, 환각률 모니터링
- **공급망·서드파티**: 외부 LLM·도구의 데이터 유출·보안 위험 평가, SBOM·계약
- **AX 인력**: AI 리터러시 교육, 시민 개발자 + 도메인 + MLE 협업 모델, 변화관리
- **KPI·ROI**: 신약 후보 발굴 속도, 공정 수율·OEE, 사무 자동화율, 컴플라이언스 적시성
