## 1. 뉴스 요약

인천시가 '공간지능 AI 행정 혁신 콘퍼런스'를 개최하고, 공간지능(Spatial Intelligence) AI를 활용한 도시 행정 서비스 혁신 방안을 발표했다. 동시에 경남도는 'AI 행정 연구회'를 본격 운영하며 지자체 행정에 AI를 체계적으로 도입하겠다고 밝혔다. 공간지능 AI는 2D 이미지나 센서 데이터로부터 3D 공간을 이해·생성·추론하는 기술로, 스탠퍼드대 Fei-Fei Li 교수가 설립한 World Labs 등이 주도하며 차세대 AI의 핵심 연구 분야로 부상하고 있다.

## 2. 핵심 개념

### 공간지능(Spatial Intelligence) AI 기술 체계

```
┌──────────────────────────────────────────────────────────────┐
│              공간지능 AI 기술 스택                              │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  [인식 계층]  2D→3D 복원 / 깊이 추정 / 객체 인식              │
│       │       SLAM, Visual Odometry, 포인트 클라우드           │
│       ▼                                                      │
│  [이해 계층]  3D 장면 그래프 생성 / 공간 관계 추론              │
│       │       시맨틱 맵핑, 공간 질의응답(Spatial QA)            │
│       ▼                                                      │
│  [생성 계층]  3D 장면 생성 / 뷰 합성 / 공간 편집               │
│       │       NeRF, 3D Gaussian Splatting, 확산 모델            │
│       ▼                                                      │
│  [행동 계층]  공간 기반 의사결정 / 경로 계획 / 로봇 내비게이션  │
│              자율주행, 드론 운용, 시설 관리 최적화               │
└──────────────────────────────────────────────────────────────┘
```

### 기존 GIS vs 공간지능 AI 비교

| 구분 | 기존 GIS/공간분석 | 공간지능(Spatial AI) |
|------|------------------|---------------------|
| 데이터 형태 | 벡터/래스터 2D 지도 | 3D 포인트클라우드, NeRF, 메시 |
| 분석 방식 | 규칙 기반 공간 질의 | 딥러닝 기반 공간 추론 |
| 3D 이해 | 2.5D(DEM/DSM) 제한적 | 완전한 3D 장면 이해 |
| 실시간성 | 배치 분석 중심 | 실시간 인식·추론 가능 |
| 생성 능력 | 없음 | 3D 장면 생성·편집 |
| 주요 표준 | OGC, ISO 19100 시리즈 | 표준화 초기 단계 |
| 활용 분야 | 지적, 도시계획, 환경 | 자율주행, 로봇, AR/VR, 도시관리 |

## 3. 기술사 출제 포인트

### 예상 문제
1. **공간지능(Spatial Intelligence) AI의 개념과 핵심 기술, 활용 분야를 설명하시오**
   - 정의: 2D 시각 정보로부터 3D 공간을 인식·이해·생성·추론하는 AI 기술
   - 핵심 기술: NeRF, 3D Gaussian Splatting, 멀티모달 공간 추론, SLAM
   - 기존 컴퓨터 비전과의 차이: 장면 수준 이해 vs 객체 수준 인식
   - 활용: 자율주행, 로봇 내비게이션, 도시 디지털 트윈, AR/VR

2. **공간 데이터 분석 플랫폼의 아키텍처를 설계하고, AI 기반 도시 관리 활용 방안을 제시하시오**
   - 데이터 레이크: 이종 공간 데이터(벡터, 래스터, 포인트클라우드) 통합 관리
   - 분석 엔진: 3D 복원, 변화 탐지, 시계열 예측 파이프라인
   - 서비스 계층: 디지털 트윈 기반 시뮬레이션, 실시간 모니터링
   - 거버넌스: 공간정보 품질 관리, 개인정보(위치정보) 비식별화

3. **NeRF와 3D Gaussian Splatting 기술을 비교하고, 디지털 트윈 구축에의 적용 방안을 설명하시오**
   - NeRF: 암묵적 표현, 높은 품질, 느린 렌더링
   - 3D GS: 명시적 표현, 실시간 렌더링, 편집 용이
   - 디지털 트윈 적용: 대규모 도시 장면 복원, 실시간 업데이트 파이프라인

### 답안 작성 키워드
`공간지능(Spatial Intelligence)`, `NeRF(Neural Radiance Fields)`, `3D Gaussian Splatting`, `SLAM`, `포인트클라우드`, `공간 데이터 레이크`, `디지털 트윈`, `PostGIS`, `OGC 표준`, `공간 추론`, `시맨틱 맵핑`, `위치정보 비식별화`
