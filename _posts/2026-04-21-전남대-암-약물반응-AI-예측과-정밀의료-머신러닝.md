## 1. 뉴스 요약

전남대학교 연구진이 암 환자별 약물 반응을 예측하는 AI 기술을 개발했다고 연합뉴스·뉴시스·오마이뉴스·네이트 등 다수 매체가 동시 보도했다. 환자 유전체·전사체 정보와 기존 임상 데이터를 학습한 AI 모델이 수십 종 항암제 중 환자에게 가장 효과적인 약물을 예측하며, 효과 없는 치료를 사전에 배제함으로써 부작용·치료 기간·의료비를 줄일 수 있다. 이는 '정밀의료(Precision Medicine)'의 핵심 과제인 약물 반응 예측(DRP, Drug Response Prediction)을 AI로 구현한 사례로, 글로벌 학술지 연구를 토대로 국내 임상 데이터와 결합한 것이 특징이다. 병원 EMR 연동을 통한 임상 적용 단계까지 계획돼 있어 학술 성과가 실제 의료현장으로 이어질지 주목된다.

## 2. 핵심 개념

### 약물 반응 예측 AI 파이프라인

```
┌──────────────────────────────────────────────────────────────┐
│           암 환자 약물 반응 예측 AI (DRP 파이프라인)               │
├──────────────────────────────────────────────────────────────┤
│ [데이터 수집·통합 계층]                                         │
│  ┌──────────────┬──────────────┬──────────────┐             │
│  │ 유전체(WGS)   │ 전사체(RNA-seq)│ 단백체         │             │
│  │ 돌연변이·CNV  │ 발현량         │ 인산화·경로    │             │
│  └──────────────┴──────────────┴──────────────┘             │
│         ↓ 통합(Multi-omics Fusion)                           │
│  임상 데이터(EMR): 나이·병기·동반질환·이전 치료 이력             │
├──────────────────────────────────────────────────────────────┤
│ [학습 데이터·참조 DB]                                          │
│  GDSC / CCLE / TCGA / PDX (Patient-Derived Xenograft)         │
│  → 세포주·환자 유래 시료 + 약물 IC50/AUC 반응값                 │
├──────────────────────────────────────────────────────────────┤
│ [모델 계층] (앙상블 + 설명가능성)                              │
│  ┌────────────┬────────────┬────────────┐                    │
│  │ GNN        │ Transformer│ Survival   │                    │
│  │ (약물·경로) │ (유전자서열)│ (생존예측) │                    │
│  └────────────┴────────────┴────────────┘                    │
│  → 약물별 반응 스코어·순위 / SHAP 피처 중요도                   │
├──────────────────────────────────────────────────────────────┤
│ [임상 적용 계층]                                              │
│  CDSS 연동(FHIR) → 의사 검토(HITL) → 처방 가이드               │
│  모니터링: 실제 반응률 수집 → 재학습 루프(MLOps)                │
└──────────────────────────────────────────────────────────────┘
```

### 정밀의료 AI의 5대 데이터 도전과제

- **고차원·소표본(High-dim Low-sample)**: 수만 개 유전자 피처 vs 수백~수천 환자 → 과적합 위험, 정규화·차원축소 필수
- **이질성(Heterogeneity)**: 종양 아형·인종·병원별 편차 → 도메인 적응(Domain Adaptation) 필요
- **결측(Missingness)**: 환자별 측정 플랫폼 상이 → 멀티모달 결측 처리(Masked Autoencoder)
- **라벨 편향**: 세포주 데이터 vs 실제 환자 임상 결과의 괴리 → 전이학습(Transfer Learning)
- **설명가능성(XAI)**: 처방 근거를 의사가 이해해야 채택 → SHAP·경로 분석(Pathway)

### 핵심 지표: IC50·AUC·ROC-AUC

아래는 약물 반응 예측에서 자주 쓰이는 평가지표다.

| 지표 | 의미 | 사용 맥락 |
|------|------|-----------|
| IC50 | 세포 증식 50% 억제 농도 | 약물 감수성 정량 |
| AUC(Dose-Response) | 용량-반응 곡선 하부 면적 | 반응 강도 연속값 |
| ROC-AUC | 분류 성능(반응 vs 비반응) | 이진 반응 예측 |
| Concordance Index | 생존 순위 정확도 | 생존 예측 모델 |

## 3. 기술사 출제 포인트

### 논술형 예상 주제

- 정밀의료 구현을 위한 멀티오믹스 AI 예측 모델 설계 방안 및 데이터 거버넌스 전략
- 임상 의사결정지원(CDSS)에서 AI 약물 추천 모델의 신뢰성·설명가능성 확보 방안
- 소표본·고차원 의료 데이터에서의 과적합 방지 및 모델 일반화 기법 비교

### 단답형 핵심 키워드

- **DRP(Drug Response Prediction)**: 약물 반응 예측 AI의 총칭
- **멀티오믹스(Multi-omics)**: 유전체·전사체·단백체·대사체 등 다층 오믹스 통합 분석
- **SHAP(SHapley Additive exPlanations)**: 모델 예측에 각 피처 기여도를 정량화하는 XAI 기법
- **전이학습(Transfer Learning)**: 도메인 A에서 학습한 지식을 도메인 B로 이전하는 학습 전략
- **연합학습(Federated Learning)**: 데이터를 중앙에 모으지 않고 분산 학습하는 기법, 의료 데이터 공유 제약 해법
- **GNN(Graph Neural Network)**: 약물-단백질 상호작용망·유전자 경로 모델링에 적합

### 비교·대조 포인트

아래는 전통적 임상시험 기반 처방과 AI 기반 정밀의료 처방의 차이점 비교다.

| 구분 | 전통 처방 | AI 기반 정밀의료 |
|------|----------|------------------|
| 기준 | 평균적 임상시험 결과 | 개별 환자 분자 프로파일 |
| 의사결정 | 가이드라인 중심 | 데이터 기반 개인화 |
| 시행착오 | 순차적 약물 시도 | 사전 반응 예측 후 선택 |
| 데이터 | 임상지표 중심 | 오믹스+EMR 통합 |
| 검증 | RCT 3상 | RWE + MLOps 지속검증 |
| 한계 | 개인 편차 미반영 | 소표본·일반화 과제 |
