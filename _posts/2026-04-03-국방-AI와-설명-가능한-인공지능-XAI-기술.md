## 1. 뉴스 요약

미국-이란 군사 충돌 사례 분석에서 AI 기반 지휘 결심이 의사결정 속도를 100배 이상 향상시킨 것으로 나타났다. 국방 AX 토론회에서는 '설명·신뢰·통제 가능한 국방 AI'의 필요성이 강조되었으며, 전군 호환 체계 구축이 요구되고 있다. 국방혁신기술보안협회와 성균관대 인공지능융합원은 국방 AI 기술 발전 및 보안 협력 MOU를 체결했다. 국방 AI가 실전에 투입되면서, 설명 가능한 인공지능(XAI: eXplainable AI)은 AI의 판단 근거를 투명하게 제시하여 지휘관의 신뢰 기반 의사결정을 지원하는 핵심 기술로 부상하고 있다.

## 2. 핵심 개념

### XAI(eXplainable AI) 개념과 필요성

```
┌─────────────────────────────────────────────────────┐
│              AI 모델의 설명 가능성 스펙트럼           │
├────────────────────┬────────────────────────────────┤
│   해석 가능 모델    │     설명 가능 모델 (XAI)       │
│  (Interpretable)   │     (Explainable)              │
├────────────────────┤────────────────────────────────┤
│ • 의사결정트리      │ • 블랙박스 모델 + 설명 기법    │
│ • 선형회귀         │ • 딥러닝 + LIME/SHAP           │
│ • 규칙 기반 시스템  │ • CNN + Grad-CAM              │
│ (모델 자체가 투명)  │ (사후 설명 기법 적용)          │
├────────────────────┴────────────────────────────────┤
│  정확도 ←──────── Trade-off ──────────→ 설명 가능성  │
│  (복잡한 모델일수록 정확하나 설명 어려움)              │
└─────────────────────────────────────────────────────┘
```

### XAI 주요 기법 분류

| 분류 기준 | 유형 | 대표 기법 | 특징 |
|----------|------|----------|------|
| 적용 범위 | 글로벌 설명 | 피처 중요도, PDP | 모델 전체 동작 패턴 설명 |
| | 로컬 설명 | LIME, SHAP, Anchor | 개별 예측 결과 설명 |
| 적용 시점 | 사전(Ante-hoc) | 해석 가능 모델 설계 | 모델 구조 자체가 투명 |
| | 사후(Post-hoc) | LIME, SHAP, Grad-CAM | 기학습 모델에 설명 추가 |
| 모델 의존성 | 모델 무관(Agnostic) | LIME, SHAP, PDP | 어떤 모델에든 적용 가능 |
| | 모델 특화(Specific) | Grad-CAM, Attention | 특정 모델 구조에 최적화 |

### 국방 AI에서 XAI 적용 프레임워크

```
┌─────────────────────────────────────────────────────┐
│        국방 AI 의사결정 지원 체계 (DARPA XAI)        │
├─────────────────────────────────────────────────────┤
│                                                     │
│  [센서 데이터]  →  [AI 분석 엔진]  →  [XAI 설명]     │
│  • 위성영상       • 표적 식별        • 판단 근거      │
│  • 신호정보       • 위협 평가        • 불확실성 수준   │
│  • 통신감청       • 행동 예측        • 대안 제시       │
│                                      ↓              │
│                              [지휘관 의사결정]        │
│                              • 설명 기반 신뢰 판단    │
│                              • AI 권고 수용/거부      │
│                              • 인간 통제 유지(HITL)   │
├─────────────────────────────────────────────────────┤
│  XAI 요구사항 (국방 특수성)                          │
│  ① 실시간성: 전투 상황 ms 단위 설명 생성             │
│  ② 다계층 설명: 지휘관/참모/운용자 수준별 설명        │
│  ③ 불확실성 표현: 확률적 신뢰도 + 근거 데이터 제시   │
│  ④ 적대적 강건성: 적 기만에 대한 AI 판단 견고성      │
│  ⑤ 감사 추적성: 모든 AI 판단의 사후 검증 가능        │
└─────────────────────────────────────────────────────┘
```

### XAI 신뢰성 평가 지표

| 평가 지표 | 설명 | 측정 방법 |
|----------|------|----------|
| 충실도(Fidelity) | 설명이 원래 모델의 동작을 얼마나 정확히 반영하는가 | 설명 모델과 원본 모델 출력 일치도 |
| 안정성(Stability) | 유사한 입력에 대해 일관된 설명을 제공하는가 | 입력 변화 대비 설명 변화 비율 |
| 이해도(Comprehensibility) | 사용자가 설명을 쉽게 이해할 수 있는가 | 사용자 실험 기반 평가 |
| 완전성(Completeness) | 모델의 모든 중요 동작을 설명하는가 | 설명 범위 대비 모델 행동 커버리지 |

## 3. 기술사 출제 포인트

### 예상 문제
- **설명 가능한 인공지능(XAI)의 개념, 주요 기법(LIME, SHAP, Grad-CAM)을 비교 설명하고, 미션크리티컬 시스템에서의 적용 방안을 논하시오**
- **AI 모델의 설명 가능성(Explainability)과 해석 가능성(Interpretability)의 차이를 설명하고, 정확도와의 Trade-off 해결 방안을 논하시오**
- **DARPA XAI 프로그램의 목표와 접근 방식을 설명하고, 공공·국방 분야 AI 신뢰성 확보 전략을 제시하시오**

### 답안 작성 키워드
- XAI = eXplainable AI, DARPA 2017년 프로그램 시작
- LIME(Local Interpretable Model-agnostic Explanations)
- SHAP(SHapley Additive exPlanations): 게임이론 기반
- Grad-CAM: CNN 시각화, 클래스 활성화 맵
- HITL(Human-In-The-Loop): 인간 통제 기반 AI 운용
- EU AI Act: 고위험 AI 시스템 설명 의무화
- 정확도-설명가능성 Trade-off와 해결 전략

### XAI 적용 분야별 요구사항

| 적용 분야 | 핵심 요구사항 | XAI 기법 |
|----------|-------------|---------|
| 국방·군사 | 실시간 판단 근거, 적대적 강건성 | Attention + SHAP |
| 의료·헬스케어 | 진단 근거 투명성, 규제 준수 | LIME + Grad-CAM |
| 금융·신용평가 | 공정성, 차별 방지, 감사 추적 | SHAP + PDP |
| 자율주행 | 실시간 의사결정 설명, 사고 원인 분석 | Grad-CAM + Attention |
| 공공행정 | 시민 권리 보호, 이의 제기 근거 | LIME + 규칙 추출 |

## 4. 관련 토픽 연계

| 연계 토픽 | 연관성 |
|----------|--------|
| AI 거버넌스·윤리 | XAI는 AI 윤리·규제(EU AI Act)의 핵심 기술 요건 |
| AI 공정성(Fairness) | 모델 편향 탐지·해소를 위한 XAI 활용 |
| 적대적 공격(Adversarial Attack) | XAI로 적대적 샘플 탐지 및 모델 강건성 검증 |
| 연합학습(Federated Learning) | 분산 학습 환경에서의 설명 가능성 확보 |
| 미션크리티컬 AI | 고신뢰 AI 시스템에서 XAI 필수 요건 |
| 디지털 포렌식 | AI 판단 감사 추적(Audit Trail) 기술 |
| 강화학습(RL) | 정책 기반 의사결정의 설명 가능성 확보 |
