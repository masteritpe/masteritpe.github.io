### 비선형 활성화 함수(Non Linear Activation Function)

### 메타휴리스틱스(Metaheuristics)

**[리드]**  

인공지능(AI)에서 최적화 문제 해결, 메타휴리스틱스  

**[개념]**  

인공지능 모델등의 학습 과정에서 최적의 솔루션을 찾기 위해 완전 탐색(Exhaustive Search) 대신 근사 최적해(Approximate Solution)를 찾는 알고리즘  
→딥러닝(DL)에서 하이퍼파라미터 튜닝, 피쳐 선택 등에 적용  

**[상세설명 및 해결방안]**  

메타휴리스틱스의 주요 알고리즘 상세 설명  

![image](https://cafeptthumb-phinf.pstatic.net/MjAyNTAzMTlfNzQg/MDAxNzQyMzM5MDgyNjU0.x2YS8dpiN0UY1FY7lr7hzwegya3GfgufZkKzaJGRFIYg.69vHir7mdpdQrhOB8x9dFjwJaaCSZi8JaQ92Pk-ohFMg.PNG/003.png?type=w1600)  

-&gt; 특정 패턴 없이 확률적 요소를 포함하여 최적해를 탐색(확률적 방법론)  

-&gt; 지역 최적해를 탈출하여 더 나은 해를 찾는 기능 포함  

메타휴리스틱스의 주요 활용 분야 제시  

- 경로 최적화: 차량 경로 문제(VRP), 외판원 문제(TSP) 해결  

- 네트워크 설계: 통신망, 전력망, 물류 네트워크 최적화  

- 머신러닝 하이퍼파라미터 최적화: 신경망 구조 및 학습률 최적화  

- 생산 스케줄링: 공장 생산 일정, 작업 공정 최적화  

-&gt; 추가적으로, 금융 포트폴리오 최적화, 유전자 서열 분석 등에도 활용

### SVM(Support Vector Machine)의 하드마진과 소프트마진 비교

**[리드]**  

데이터 간 최대 마진을 가지는 결정경계 활용, SVM  

**[개념]**  

데이터 포인트를 고차원 공간에서 분리하는 최적의 결정경계(Decision Boundary)를 찾아주는 지도 학습(Supervised Learning) 알고리즘  
- 분류문제에 대한 접근 방식은 소프트마진과 하드마진으로 구분  
- 데이터가 완벽하게 선형 분리 가능 → 하드 마진 사용  
- 노이즈가 있거나 완전한 분리가 어려움 → 소프트 마진 사용 (일반적으로 사용)  

**[상세설명 및 해결방안]**  

SVM의 하드마진과 소프트마진 비교

- 실제 데이터는 대부분 완벽하게 선형 분리되지 않으므로, 소프트 마진 SVM이 자주 사용  

- 하드 마진은 제약 조건이 강함 → 일반적으로 적용 범위가 제한적  

- 소프트 마진은 일부 데이터를 오차로 허용 → 실제 데이터에 유용하며 일반적으로 더 많이 사용됨  

- 소프트 마진은 오차를 얼마나 허용할지 조정하는 'C' 하이퍼파라미터 사용 → C 값이 크면 하드 마진에 가까워지고, 작으면 더 많은 오차 허용

### PCA(Principal Component Analysis)와 LDA(Linear Discriminant Analysis) 비교

**[리드]**  

고차원 데이터 차원축소 기법, PCA와 LDA  

**[개념]**  

PCA : 데이터의 분산(Variance)을 최대한 보존하면서 고차원 데이터를 저차원으로 축소하는 선형 차원 축소 기법  

LDA : 독립변수들의 측정값에 따라 데이터가 어느 집단에 속할 것인가에 대해 판별하는 분석 방법  

-&gt; PCA는 고차원 공간의 표본들을 선형 연관성이 없는 저 차원 공간(주성분)의 표본으로 변환하는 비지도 학습이며, LDA는 분산 대비 평균의 차이를 최대화하는 직선을 발견하는 지도학습에 해당됨  

**[상세설명 및 해결방안]**  

PCA와 LDA의 상세 비교

- (QDA의 극복방안) 각클래스별로 서로 다른 공분산 행렬을 허용, 비선형(이차) 판별경계 형성 / 이차 (곡선 가능)

### 연관규칙(Association Rule)의 분석을 위한 3가지 척도

**[리드]**  

데이터 집합 내의 연관관계 발견, 연관규칙

### 강화학습(Reinforcement Learning)가. 강화학습의 정의와 구성요소나. 강화학습의 종류다. 탐색(Exploration)과 활용(Exploitation)의 균형 문제

**[리드]**  

보상기반 최적의 의사결정 학습, 강화학습  

**[개념]**  

학습 요소 측면 정의 : 데이터의 상태(State)을 인식하고 이에 반응한 행위(Action)에 대하여 환경으로부터 받는 포상(Reward)을 학습하여 행위에 대한 포상을 최적화하는 정책(Model)을 찾는 기계학습  

최적화 기반 정의 : 반복적인 결정 및 착오와 경험에 기반하여 상태의 최적의 행동을 조금씩 학습하는 알고리즘  

-&gt; 강화학습은 상태, 행동, 보상, 정책을 기반으로 학습 방식과 최적화 수행  

**[상세설명 및 해결방안]**  

보상기반 최적의 의사결정 학습, 강화학습의 정의와 구성요소  

가. 학습요소와 최적화 기반, 강화학습의 정의  

- 학습 요소 측면 정의 : 데이터의 상태(State)을 인식하고 이에 반응한 행위(Action)에 대하여 환경으로부터 받는 포상(Reward)을 학습하여 행위에 대한 포상을 최적화하는 정책(Model)을 찾는 기계학습  

- 최적화 기반 정의 : 반복적인 결정 및 착오와 경험에 기반하여 상태의 최적의 행동을 조금씩 학습하는 알고리즘  

-&gt; 강화학습은 상태, 행동, 보상, 정책을 기반으로 학습 방식과 최적화 수행  

나. 강화학습의 구성요소 상세 설명  

![image](https://cafeptthumb-phinf.pstatic.net/MjAyNTAzMjNfMTY0/MDAxNzQyNzEwMDYwOTYy.bHE0Y9kwUQjerS1hcgrlr1e79iF-YLmtBZPj9NX-7Eog.S5KvA081l-K3ti32hKE4czzgQA3rA6ykrsSIRymeSdQg.PNG/020.png?type=w1600)  

- Q함수는 가치 기반 학습에서 특정 상태에서 특정 행동을 했을 때 기대되는 보상을 평가하는 요소이나, 강화학습의 진화론적 측면에서 의의를 부여함  

정책, 모델, 온/오프 기반강화학습의 종류 설명  

정책 기반 vs 가치 기반  

- 정책 기반 (Policy-Based): 행동을 직접 최적화 (예: REINFORCE)  

- 가치 기반 (Value-Based): 상태-행동 가치(Q-value)를 학습하여 최적 정책 유도 (예: Q-learning, DQN)  

- 혼합 방식 (Actor-Critic): 정책과 가치를 함께 학습  

모델 기반 vs 모델 프리  

- 모델 기반 (Model-Based): 환경의 동적 모델을 학습하여 미래 예측 가능 (예: Dyna-Q)  

- 모델 프리 (Model-Free): 환경 모델 없이 직접 경험을 통해 학습 (예: Q-learning, PPO, DDPG)  

온정책 vs 오프정책  

- 온정책 (On-Policy): 현재 사용 중인 정책을 기반으로 학습 (예: SARSA, PPO)  

- 오프정책 (Off-Policy): 과거의 경험 데이터도 활용하여 학습 (예: Q-learning, DQN)  

--&gt; 강화학습은 학습 방식, 환경 모델 사용 여부, 정책 적용 방식, 행동 유형에 따라 분류  

탐색(Exploration)과 활용(Exploitation)의 균형 문제  

- 탐색(Exploration): 새로운 행동을 시도하여 더 나은 보상을 찾는 과정  

- 활용(Exploitation): 현재까지 학습한 정보를 활용하여 최적의 보상을 얻는 과정  

&lt;탐색과 활용의 균형 문제&gt;  

- 탐색 부족 → 최적의 행동을 찾지 못하고 로컬 최적해(Local Optimum)에 갇힘  

- 활용 부족 → 학습 속도가 느려지고, 비효율적인 행동을 반복할 가능성 증가  

&lt;해결방법&gt;  

- ε-탐욕 정책(ε-Greedy Policy): 확률 ε로 무작위 탐색, (1-ε) 확률로 최적 행동 선택  

- 볼츠만 탐색(Boltzmann Exploration): 행동의 선택 확률을 보상 기대값 기반으로 설정  

- 어퍼 컨피던스 바운드(UCB, Upper Confidence Bound): 행동의 불확실성을 고려하여 탐색  

- 베이지안 접근법: 확률적 모델을 사용하여 탐색과 활용을 동적으로 조정  

- 딥 강화학습에서의 적용: DQN에서는 ε-탐욕 정책, PPO·SAC에서는 엔트로피 보상 추가 등 활용  

-&gt; 탐색과 활용의 균형을 유지하는 것은 강화학습의 성능을 결정하는 핵심 요소이며, 다양한 전략을 활용해 최적의 정책  

### 머신 러닝(Machine Learning)에서 활용되는 앙상블(Ensemble)기법

**[개념]**

**[상세설명 및 해결방안]**

- 단일 모델 n 개 -&gt; 예측 정확성 : 고( 그래프로 표현)
