## 1. 뉴스 요약

글로벌 의료기기 업체 스트라이커가 사이버공격을 받아 주문·출하 시스템이 마비되었고, 복구에 총력을 기울이고 있다. 국내에서는 행정안전부의 해킹대응 인력 3분의 2가 공석인 것으로 드러나 정보보안 체계에 심각한 구멍이 있음이 확인되었다. 부산정보산업진흥원은 지역 기업의 사이버 공격 대응 역량 강화를 위한 교육 프로그램을 시작했고, 정보협은 정기총회에서 지방정부·공공 보안관리체계 고도화 필요성을 강조했다.

## 2. 핵심 개념

### 침해사고 대응 프로세스 (NIST CSF 2.0)
```
식별(Identify) → 보호(Protect) → 탐지(Detect) → 대응(Respond) → 복구(Recover) → 거버넌스(Govern)
```

### CSIRT(Computer Security Incident Response Team)
- **역할**: 침해사고 예방, 탐지, 분석, 대응, 복구
- **구성**: SOC(보안관제센터) + CERT(침해사고대응팀) + 포렌식팀
- **국내 체계**: KISA(한국인터넷진흥원) → 분야별 ISAC → 기관 CSIRT

### 스트라이커 사이버공격 사례 분석

| 항목 | 내용 |
|------|------|
| 공격 대상 | 글로벌 의료기기 제조사(공급망) |
| 피해 범위 | 주문·출하 시스템 마비 |
| 시사점 | OT(운영기술) 환경 보안의 중요성 |
| 대응 | BCP(업무연속성계획) 기반 복구 |

### 보안 인력 부족 문제
행안부 사례가 보여주듯, **정보보안 전문인력 확보**는 기술적 과제가 아닌 **조직·제도적 과제**다.
- 공공기관 보안인력 충원율 저조
- 민간 대비 낮은 처우
- 보안 전문가 수급 불균형

## 3. 기술사 출제 포인트

> **예상 문제**: 침해사고 대응체계(CSIRT)의 구성과 역할을 설명하고, 효과적인 침해사고 대응을 위한 개선 방안을 제시하시오.

- **킬체인(Kill Chain)**: 록히드마틴의 사이버 킬체인 7단계 — 정찰→무기화→전달→익스플로잇→설치→C2→목표달성
- **MITRE ATT&CK**: 공격자 전술·기술·절차(TTP) 매트릭스 기반 위협 모델링
- **BCP/DRP**: 업무연속성계획과 재해복구계획의 연계
- **공급망 보안**: 의료기기 등 OT 환경에서의 보안 위협 증가
- **보안 거버넌스**: CISO 역할, 보안 조직 체계, 인력 양성 정책

## 4. 관련 토픽 연계

- **IT경영**: 정보보호 관리체계(ISMS-P), 위험관리, BCP
- **네트워크**: IDS/IPS, NDR, 네트워크 포렌식
- **시스템구조**: OT 보안, 산업제어시스템(ICS) 보안
- **소프트웨어공학**: DevSecOps, 시큐어코딩
