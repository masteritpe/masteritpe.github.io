## 1. 뉴스 요약

AI 인프라 경쟁이 본격화되고 있다. 엔비디아는 '베라 루빈' 초대형 AI 슈퍼컴퓨터 아키텍처를 공개했고, 머스크는 전세계 AI칩 생산량의 50배 규모 '테라팹' 건설을 선언했다. 국내에서는 IITP가 4대 분야에서 에이전틱 AI 기술개발 과제당 45억원을 지원하며, 네이버는 자체 LLM 실험을 종료하고 쇼핑 에이전트 강화로 전략을 전환했다. 하이퍼엑셀은 생성형 AI 전용 LPU(Language Processing Unit)로 AI 반도체 시장에 도전장을 내밀었다.

## 2. 핵심 개념

### 에이전틱 AI(Agentic AI)
- **정의**: 사람의 개입 없이 자율적으로 목표를 설정하고, 계획을 수립하며, 도구를 사용하여 작업을 수행하는 AI 시스템
- **기존 생성형 AI와 차이**: 단순 응답 생성 → 자율적 의사결정 + 행동 실행
- **핵심 구성**: 계획(Planning) → 도구 사용(Tool Use) → 메모리(Memory) → 반성(Reflection)

### AI 인프라 스택
```
┌─────────────────────────────────┐
│  애플리케이션 (에이전트, 챗봇)     │
├─────────────────────────────────┤
│  프레임워크 (LangChain, AutoGen) │
├─────────────────────────────────┤
│  모델 (LLM, SLM, 멀티모달)       │
├─────────────────────────────────┤
│  학습/추론 플랫폼                 │
├─────────────────────────────────┤
│  AI 반도체 (GPU, TPU, LPU, NPU)  │
├─────────────────────────────────┤
│  데이터센터 / 슈퍼컴퓨터          │
└─────────────────────────────────┘
```

### AI 전용 반도체 동향

| 유형 | 대표 제품 | 특징 |
|------|----------|------|
| GPU | NVIDIA H100/B200 | 범용 병렬 연산, 학습+추론 |
| TPU | Google TPU v5p | 텐서 연산 최적화 |
| LPU | Groq, 하이퍼엑셀 | 추론 특화, 저지연 |
| NPU | 퀄컴, 삼성 | 온디바이스 AI |

### 네이버 AI 전략 전환의 시사점
자체 LLM 개발 → 쇼핑 에이전트 강화로의 전환은 **LLM 자체보다 응용(Application Layer)에서 차별화**하겠다는 전략. 기술사 관점에서 AI 기술 선택 시 Build vs Buy 의사결정의 좋은 사례.

## 3. 기술사 출제 포인트

> **예상 문제**: 에이전틱 AI의 개념과 기존 생성형 AI와의 차이점을 설명하고, 기업 도입 시 고려사항을 서술하시오.

- **ReAct 패턴**: Reasoning + Acting을 반복하는 에이전트 아키텍처
- **RAG(Retrieval-Augmented Generation)**: 외부 지식 기반을 활용한 LLM 응답 품질 향상
- **멀티 에이전트 시스템**: 복수의 AI 에이전트가 협업하여 복잡한 문제를 해결
- **AI 인프라 비용 최적화**: GPU 클러스터 vs 클라우드 AI 서비스 vs 경량 모델(SLM)
- **AI 거버넌스**: 자율적 AI 시스템의 위험 관리, 설명가능성(XAI), 편향 감지

## 4. 관련 토픽 연계

- **소프트웨어공학**: AI 기반 개발(AI-Assisted Development), MLOps, LLMOps
- **시스템구조**: GPU 클러스터 아키텍처, 분산 학습, 모델 병렬화
- **IT경영**: AI 투자 ROI, AI 전략 수립, Build vs Buy 의사결정
- **정보보안**: AI 보안 위협(Prompt Injection, Data Poisoning)
