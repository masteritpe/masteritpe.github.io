## 1. 뉴스 요약

LG유플러스와 LG전자가 6G 핵심 기술인 시맨틱 통신(Semantic Communication)과 PQC(Post-Quantum Cryptography) 기반 선행 연구 및 국제 표준화 협력에 나선다. 시맨틱 통신은 기존 비트 단위 데이터 전송 방식에서 벗어나 '의미(Semantic)' 단위로 정보를 압축·전송하는 차세대 통신 패러다임으로, AI/딥러닝 모델을 송수신 양단에 배치하여 통신 효율을 혁신적으로 향상시킨다. 6G 시대의 초저지연·초대용량 요구사항을 충족하기 위한 핵심 기술로 글로벌 표준화 경쟁이 본격화되고 있다.

## 2. 핵심 개념

### 시맨틱 통신 vs 전통적 통신 아키텍처

```
[전통적 통신 (Shannon 모델)]
  송신자 → 소스코딩 → 채널코딩 → 변조 → 채널 → 복조 → 채널디코딩 → 소스디코딩 → 수신자
  (비트 단위 정확한 복원 목표)

[시맨틱 통신 (Semantic 모델)]
  송신자 → 의미추출(AI) → 시맨틱코딩 → 채널 → 시맨틱디코딩 → 의미복원(AI) → 수신자
  (의미 단위 정확한 전달 목표)
```

### 시맨틱 통신 시스템 계층 구조

| 계층 | 기존 통신 | 시맨틱 통신 | 핵심 기술 |
|------|----------|-----------|----------|
| 기술적 계층(Level A) | 비트 정확도(BER) | 심볼 전송 정확도 | 채널코딩, 변조 |
| 의미적 계층(Level B) | 해당 없음 | 의미 정확도(Semantic Accuracy) | 딥러닝 인코더/디코더, 지식그래프 |
| 효과적 계층(Level C) | 해당 없음 | 목적 달성도(Effectiveness) | 태스크 지향 통신, 강화학습 |

### 시맨틱 통신 핵심 구성요소

```
┌─────────────────────────────────────────────────────┐
│           시맨틱 통신 시스템 아키텍처                  │
├──────────────────┬──────────────────────────────────┤
│   송신단(Tx)      │         수신단(Rx)                │
├──────────────────┤──────────────────────────────────┤
│ 시맨틱 인코더     │ 시맨틱 디코더                     │
│ ┌──────────────┐ │ ┌──────────────────────────────┐ │
│ │ 지식베이스(KB)│ │ │ 지식베이스(KB)                │ │
│ │ - 공유 배경지식│ │ │ - 동일 KB 보유               │ │
│ ├──────────────┤ │ ├──────────────────────────────┤ │
│ │ 의미 추출기   │ │ │ 의미 복원기                   │ │
│ │ - Transformer │ │ │ - Transformer Decoder        │ │
│ │ - CNN/RNN     │ │ │ - 생성 모델                   │ │
│ ├──────────────┤ │ ├──────────────────────────────┤ │
│ │ 채널 인코더   │ │ │ 채널 디코더                   │ │
│ │ - JSCC       │ │ │ - Joint Source-Channel       │ │
│ └──────────────┘ │ └──────────────────────────────┘ │
│        ↓          │          ↑                       │
│    [무선 채널]  ───────────→                          │
└──────────────────┴──────────────────────────────────┘
```

### 시맨틱 통신 핵심 기술 요소

| 기술 요소 | 설명 | 적용 사례 |
|----------|------|----------|
| JSCC(Joint Source-Channel Coding) | 소스코딩과 채널코딩을 AI로 통합 최적화 | 이미지·영상 전송 시 압축률 10배 이상 향상 |
| 지식베이스(Knowledge Base) | 송수신 공유 배경지식으로 전송량 최소화 | 동일 언어모델 공유로 텍스트 90% 압축 |
| 태스크 지향 통신(Task-Oriented) | 목적에 필요한 정보만 선별 전송 | 자율주행 차량 간 객체 정보만 교환 |
| 의미 유사도 메트릭 | BER 대신 의미적 정확도로 성능 평가 | BLEU, Sentence Similarity 등 |

## 3. 기술사 출제 포인트

### 예상 문제
- **시맨틱 통신(Semantic Communication)의 개념과 기존 Shannon 통신 모델과의 차이점을 설명하고, 6G 네트워크에서의 적용 방안을 논하시오**
- **JSCC(Joint Source-Channel Coding)의 원리와 AI 기반 시맨틱 통신에서의 역할을 설명하시오**
- **6G 네트워크의 핵심 기술 요소(시맨틱 통신, PQC, AI 네이티브)를 비교 설명하시오**

### 답안 작성 키워드
- Shannon 3계층 모델(기술적·의미적·효과적)
- JSCC(Joint Source-Channel Coding): 소스/채널 코딩 AI 통합
- 지식베이스(KB) 공유를 통한 전송 효율화
- 태스크 지향 통신(Task-Oriented Communication)
- 의미 유사도 메트릭(Semantic Similarity Metric)
- DeepSC(Deep Semantic Communication) 프레임워크

### 시맨틱 통신 성능 비교

| 비교 항목 | 전통적 통신 | 시맨틱 통신 | 개선 효과 |
|----------|-----------|-----------|----------|
| 전송 데이터량 | 원본 크기 비례 | 의미 단위 압축 | 10~100배 감소 |
| 저SNR 환경 성능 | 급격히 저하 | 의미 보존으로 강건 | Cliff Effect 완화 |
| 대역폭 효율 | 고정 코딩률 | 적응형 시맨틱 코딩 | 3~10배 향상 |
| 다중 사용자 | 직교 분할 필요 | 의미 기반 다중접속 | 수용 용량 증대 |

## 4. 관련 토픽 연계

| 연계 토픽 | 연관성 |
|----------|--------|
| 6G 네트워크 아키텍처 | 시맨틱 통신이 6G의 핵심 기술 요소로 채택 |
| AI 네이티브 네트워크 | 송수신 AI 모델 기반 통신 최적화 |
| PQC(Post-Quantum Cryptography) | 시맨틱 통신 보안을 위한 양자안전 암호체계 |
| 자율주행 V2X | 시맨틱 통신의 대표적 적용 시나리오 |
| 디지털 트윈 | 실시간 상태 동기화를 위한 효율적 데이터 전송 |
| Edge AI | 에지에서의 시맨틱 인코딩/디코딩 처리 |
| 지식그래프 | 시맨틱 통신 지식베이스 구축 기반 기술 |
