## 1. 뉴스 요약

한화비전이 글로벌 AI 반도체 기업 암바렐라(Ambarella)와 차세대 AI 영상보안 기술 개발 협력을 발표했다. 엣지 디바이스에서 직접 AI 추론을 수행하는 온디바이스 AI 기술이 핵심이다. 또한 Tosci AI·Tosun은 차량 네트워크 분석 및 보안 솔루션에 Edge AI를 적용하여 실시간 이상탐지를 시연했다. 해외에서는 Edge AI가 제조업의 차세대 성장동력으로 주목받으며, Physical AI와 결합한 산업용 자율 시스템 개발이 가속화되고 있다.

## 2. 핵심 개념

### Edge AI
- **정의**: 클라우드가 아닌 엣지 디바이스(카메라, 센서, 차량 등)에서 직접 AI 모델을 실행하여 추론하는 기술
- **클라우드 AI 대비 특성**:

| 구분 | Cloud AI | Edge AI |
|------|----------|---------|
| 추론 위치 | 데이터센터 | 엣지 디바이스 |
| 지연시간 | 수백ms | 수ms |
| 네트워크 의존 | 필수 | 불필요(오프라인 가능) |
| 프라이버시 | 데이터 전송 필요 | 로컬 처리(데이터 외부 유출 없음) |
| 처리 능력 | 고성능 | 제한적(경량 모델 필요) |
| 비용 구조 | API 호출 비용 | 하드웨어 초기 비용 |

### Edge AI 시스템 아키텍처
```
[센서/카메라] → [전처리 모듈] → [NPU/AI 가속기] → [추론 결과]
                                      │                    │
                              [모델 저장소]          [액추에이터/알림]
                                      │
                              [클라우드 연동]
                              (모델 업데이트, 재학습)
```

### AI 전용 프로세서(NPU) 아키텍처

| 프로세서 | 특징 | 주요 연산 | 전력 효율 |
|----------|------|----------|----------|
| CPU | 범용, 순차 처리 | 제어 흐름 | 낮음(AI 기준) |
| GPU | 병렬 처리, 부동소수점 | 행렬 곱셈 | 중간 |
| NPU | AI 추론 특화 | INT8/FP16 행렬 연산 | 높음 |
| FPGA | 재구성 가능 하드웨어 | 맞춤 연산 파이프라인 | 높음 |

### 모델 경량화 기법
- **양자화(Quantization)**: FP32 → INT8/INT4로 모델 파라미터 정밀도 축소, 모델 크기 4~8배 감소
- **프루닝(Pruning)**: 중요도 낮은 가중치/뉴런 제거, 연산량 감소
- **지식 증류(Knowledge Distillation)**: 대형 Teacher 모델의 지식을 소형 Student 모델로 전이
- **모델 분할(Model Partitioning)**: 모델의 일부는 엣지에서, 나머지는 클라우드에서 실행

### Edge-Cloud 하이브리드 아키텍처
```
[Edge Layer]                    [Cloud Layer]
┌─────────────┐                ┌──────────────┐
│ 실시간 추론  │   ← 모델 배포 ─ │ 모델 학습/최적화│
│ 데이터 전처리│   ─ 메타데이터 →│ 데이터 집계    │
│ 이상 감지    │                │ 글로벌 분석    │
│ 즉각 대응    │                │ 모델 버전 관리  │
└─────────────┘                └──────────────┘
```

## 3. 기술사 출제 포인트

> **예상 문제**: Edge AI의 개념과 시스템 아키텍처를 설명하고, 온디바이스 추론을 위한 모델 경량화 기법을 비교 분석하시오.

- **NPU 아키텍처**: AI 전용 프로세서의 구조와 기존 CPU/GPU 대비 장점
- **모델 최적화**: 양자화, 프루닝, 지식 증류의 원리와 정확도-성능 트레이드오프
- **실시간 시스템**: Edge AI에서의 실시간 처리 요구사항과 WCET(최악실행시간) 보장
- **OTA 업데이트**: 엣지 디바이스의 AI 모델 원격 업데이트와 버전 관리
- **연합학습(Federated Learning)**: 엣지 데이터를 중앙 집중하지 않고 분산 학습하는 기법

## 4. 관련 토픽 연계

- **네트워크**: 5G/6G MEC(Multi-access Edge Computing), 엣지-클라우드 통신 프로토콜
- **정보보안**: 엣지 디바이스 보안(TPM, 보안 부팅), AI 모델 역공학 방어
- **AI데이터분석**: 경량 딥러닝 모델(MobileNet, EfficientNet), TinyML
- **신기술융합**: 피지컬 AI, 자율주행, 스마트팩토리에서의 Edge AI 활용
