## 1. 뉴스 요약

국방AX 토론회에서 '설명·신뢰·통제 가능한 국방 AI' 구축의 필요성이 강조되었다. 미·이란전 사례에서 AI 기반 지휘 결심이 100배 빨라진 것으로 분석되며 국방 AI의 실전 효용성이 입증되었으나, 동시에 설명가능성(Explainability)과 통제가능성(Controllability) 없이는 실전 배치가 불가능하다는 점이 지적되었다. 또한 에이전틱 AI 얼라이언스가 출범하며 민·관 협력 체계가 본격 가동되고 있어, 자율적으로 판단하는 AI 시스템의 신뢰성 확보가 핵심 과제로 부상하고 있다.

## 2. 핵심 개념

### XAI(Explainable AI) 분류 체계

```
┌───────────────────────────────────────────────────┐
│          XAI (Explainable AI) 분류                 │
├───────────────────┬───────────────────────────────┤
│  설명 시점 기준    │  설명 범위 기준               │
├───────────────────┼───────────────────────────────┤
│                   │                               │
│  ┌─────────────┐  │  ┌─────────────┐              │
│  │ Ante-hoc    │  │  │ Global      │              │
│  │ (사전 설명)  │  │  │ (전체 모델  │              │
│  │ - 본질적    │  │  │  동작 설명)  │              │
│  │   해석 가능  │  │  └─────────────┘              │
│  │ - 의사결정  │  │                               │
│  │   트리, 선형│  │  ┌─────────────┐              │
│  │   모델, 규칙│  │  │ Local       │              │
│  └─────────────┘  │  │ (개별 예측  │              │
│                   │  │  결과 설명)  │              │
│  ┌─────────────┐  │  └─────────────┘              │
│  │ Post-hoc    │  │                               │
│  │ (사후 설명)  │  │  ┌─────────────┐              │
│  │ - 블랙박스  │  │  │ Feature     │              │
│  │   모델 해석 │  │  │ Attribution │              │
│  │ - LIME,SHAP │  │  │ (특성 기여도│              │
│  │   Attention │  │  │  분석)      │              │
│  └─────────────┘  │  └─────────────┘              │
└───────────────────┴───────────────────────────────┘
```

### 주요 XAI 기법 비교

| 기법 | 유형 | 원리 | 장점 | 한계 |
|------|------|------|------|------|
| LIME | Post-hoc, Local | 입력 주변 선형 근사 모델 생성 | 모델 비의존적, 직관적 | 불안정성, 근사 오차 |
| SHAP | Post-hoc, Local/Global | Shapley 값 기반 특성 기여도 | 이론적 근거(게임이론), 일관성 | 계산 비용 높음 |
| Attention | Ante-hoc | 입력 토큰 간 가중치 시각화 | 내장형, 추가 비용 없음 | 인과관계 불명확 |
| Grad-CAM | Post-hoc, Local | 기울기 기반 활성화 맵 | 영상 분야 직관적 시각화 | CNN 한정 |
| Concept Bottleneck | Ante-hoc | 사람이 이해 가능한 개념 학습 | 개념 수준 설명 가능 | 개념 정의 필요 |
| Counterfactual | Post-hoc, Local | 결과 바꾸는 최소 입력 변경 탐색 | 직관적("~였다면") | 다수 반사실 존재 |

### AI 신뢰성 수준별 분류

| 신뢰 수준 | 설명 | AI 자율성 | 사람 역할 | 적용 사례 |
|----------|------|----------|----------|----------|
| Level 0: 인간 전담 | AI 미사용 | 없음 | 전적 판단 | 수동 분석 |
| Level 1: 인간 주도 | AI 보조 제안 | 제안만 | 의사결정 주체 | 정보 분석 보조 |
| Level 2: 혼합 주도 | AI-인간 협업 | 부분 자율 | 승인/거부 | 표적 식별 보조 |
| Level 3: AI 주도 | AI 주도, 인간 감독 | 높은 자율 | 모니터링/개입 | 사이버 방어 |
| Level 4: 완전 자율 | AI 독립 수행 | 완전 자율 | 사후 감사 | 미사일 방어(제한적) |

## 3. 기술사 출제 포인트

### 예상 문제
- **설명가능한 인공지능(XAI)의 개념과 주요 기법(LIME, SHAP, Attention)을 비교 설명하시오**
- **미션크리티컬 시스템에서 AI 신뢰성 확보를 위한 XAI 적용 방안을 논하시오**
- **AI 시스템의 신뢰성 수준(Human-in-the-Loop, Human-on-the-Loop, Human-out-of-the-Loop)을 비교하고, 각 수준별 적용 기준을 설명하시오**

### 답안 작성 키워드
- XAI = Ante-hoc(본질적 해석) + Post-hoc(사후 설명)
- LIME: 지역 선형 근사, SHAP: Shapley 기반 특성 기여도
- DoD RAIT 5원칙: Responsible, Equitable, Traceable, Reliable, Governable
- Human-in-the-Loop → Human-on-the-Loop → Human-out-of-the-Loop
- AI TRUSTworthiness: 공정성, 투명성, 견고성, 개인정보보호

### XAI 기법 선택 기준

| 기준 | LIME | SHAP | Grad-CAM | Counterfactual |
|------|------|------|----------|---------------|
| 데이터 유형 | 정형/텍스트 | 정형/텍스트 | 이미지 | 정형 |
| 모델 의존성 | 비의존 | 비의존 | CNN 한정 | 비의존 |
| 설명 대상 | 개별 예측 | 개별/전체 | 개별 예측 | 개별 예측 |
| 계산 비용 | 중간 | 높음 | 낮음 | 높음 |
| 실시간 적용 | 가능 | 제한적 | 가능 | 제한적 |

## 4. 관련 토픽 연계

| 연계 토픽 | 연관성 |
|----------|--------|
| AI 거버넌스 | XAI는 AI 거버넌스의 투명성·책임성 구현 핵심 기술 |
| 에이전틱 AI | 자율 판단 에이전트의 의사결정 설명가능성 필수 |
| AI 윤리(AI Ethics) | 공정성·편향 탐지를 위한 XAI 활용 |
| 연합학습(Federated Learning) | 분산 모델의 설명가능성 확보 과제 |
| AI 보안 | 적대적 공격(Adversarial Attack) 탐지에 XAI 활용 |
| 디지털 트윈 | 시뮬레이션 기반 AI 의사결정 설명 |
| 미션크리티컬 시스템 | 고신뢰 요구 환경에서의 XAI 필수 적용 |
