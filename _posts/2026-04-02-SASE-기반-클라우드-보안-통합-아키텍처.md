## 1. 뉴스 요약

메가존클라우드가 글로벌 보안 기업 지스케일러(Zscaler)와 파트너십을 체결하여 SASE(Secure Access Service Edge)·HALO 기반 제로트러스트 클라우드 보안 사업을 강화한다. 동시에 HPE는 AI 확산으로 커진 분산 환경의 보안 공백에 대응하여 코어·에지·클라우드를 연결하는 복원력형 보안 아키텍처를 발표했다. 하이브리드·멀티클라우드 환경이 확산되면서, 기존 경계 기반 보안 모델의 한계를 극복하기 위한 SASE 아키텍처가 차세대 네트워크 보안의 표준으로 부상하고 있다.

## 2. 핵심 개념

### SASE vs 전통적 보안 아키텍처 비교

| 구분 | 전통적 경계 보안 | SASE 아키텍처 |
|------|-----------------|--------------|
| 보안 경계 | 데이터센터 중심 방화벽 | 클라우드 에지(PoP) 분산 |
| 접근 제어 | VPN + IP 기반 | ZTNA(ID/Context 기반) |
| 트래픽 경로 | 백홀(Backhaul) 집중 | 최적 경로 직접 접속 |
| 확장성 | 하드웨어 의존 | 클라우드 네이티브 탄력 확장 |
| 정책 관리 | 장비별 개별 관리 | 단일 클라우드 콘솔 통합 |
| 지연 시간 | 높음(중앙 집중) | 낮음(에지 처리) |
| 원격 근무 지원 | 제한적(VPN 병목) | 네이티브 지원 |
| 운영 복잡도 | 높음(멀티벤더) | 낮음(통합 플랫폼) |

### 분산 환경 복원력형 보안 계층

| 계층 | 보안 기능 | 적용 위치 |
|------|----------|----------|
| 코어(Core) | 데이터 암호화, 접근통제, 위협 인텔리전스 | 데이터센터, 프라이빗 클라우드 |
| 에지(Edge) | 실시간 트래픽 검사, IoT 보안, AI 기반 이상 탐지 | 엣지 컴퓨팅 노드, 지사 |
| 클라우드(Cloud) | CASB, SWG, ZTNA, API 보안 | 퍼블릭 클라우드, SaaS |
| 엔드포인트 | EDR/XDR, 디바이스 신뢰도 평가 | 사용자 단말, IoT 디바이스 |

## 3. 기술사 출제 포인트

### 예상 문제
- **SASE(Secure Access Service Edge)의 개념과 구성요소를 설명하고, 기존 경계 기반 보안과의 차이점을 비교하시오**
- **SSE(Security Service Edge)와 SD-WAN의 통합 관점에서 SASE 아키텍처 설계 방안을 논하시오**
- **하이브리드 클라우드 환경에서 ZTNA, CASB, SWG 기반 통합 보안 아키텍처를 설계하시오**

### 답안 작성 키워드
- SASE = SD-WAN + SSE(ZTNA + CASB + SWG + FWaaS)
- PoP(Point of Presence) 기반 글로벌 에지 보안
- 제로트러스트 원칙: Never Trust, Always Verify
- 클라우드 네이티브 보안 서비스 통합
- 사용자 중심(User-Centric) 보안 정책

### 핵심 비교 포인트

| 비교 항목 | VPN 기반 | ZTNA 기반 | SASE 통합 |
|----------|---------|----------|----------|
| 인증 방식 | 네트워크 레벨 | 애플리케이션 레벨 | 컨텍스트 기반 통합 |
| 가시성 | 터널 내부 제한적 | 앱별 세분화 | 전체 트래픽 가시성 |
| 공격 표면 | 넓음(네트워크 전체) | 축소(앱 단위) | 최소화(동적 제어) |
| 성능 | 백홀 지연 발생 | 직접 접속 가능 | 에지 최적화 |

## 4. 관련 토픽 연계

| 연계 토픽 | 연관성 |
|----------|--------|
| 제로트러스트 아키텍처(ZTA) | SASE의 핵심 보안 원칙으로 ZTNA 구현 |
| SD-WAN | SASE의 네트워크 서비스 계층, WAN 최적화 |
| 클라우드 네이티브 보안 | SASE의 클라우드 기반 보안 서비스 제공 모델 |
| 마이크로세그멘테이션 | ZTNA와 연계한 내부 네트워크 세분화 |
| DLP(Data Loss Prevention) | CASB·SWG 연동 데이터 유출 방지 |
| XDR(Extended Detection & Response) | SASE 엔드포인트 보안 연계 위협 탐지·대응 |
| API 보안 | 클라우드 서비스 간 통신 보호, CASB 연동 |
