## 1. 뉴스 요약

광주과학기술원(GIST) AI대학원 연구팀이 '핵심 감정 단서(core cue)만으로 전체 감정 흐름을 복원하는' 감정 인식 AI 기술을 개발했다고 2026년 4월 발표했다. 기존 감정 인식 모델은 대량의 프레임 단위 레이블링이 필요해 데이터 구축 비용이 높았으나, 이번 연구는 핵심 구간만 학습해도 전체 시퀀스의 감정 상태를 재구성할 수 있는 표현학습(Representation Learning) 기법을 제안해 데이터 수집·레이블링 부담을 획기적으로 절감했다. 인공지능신문·동아사이언스·전자신문·네이트 등 4개 매체가 동시 보도했으며, 국제 저명 학술지 게재 예정이다. 적용 영역은 HCI(감정 인지 챗봇·콜센터), 돌봄 로봇, 미디어 추천, 정신건강 모니터링 등 멀티모달 AI 전반에 걸친다. 핵심은 **데이터 효율(Data Efficiency) 학습 방법론**이라는 점에서 AI 일반 기술 관점에서도 의의가 크다.

## 2. 핵심 개념

### 감정 인식 AI의 기술적 도전 과제

```
기존 프레임 단위 레이블링                GIST 핵심 단서 복원 방식
┌──────────────────────────────┐      ┌──────────────────────────────┐
│ t1: 중립  t2: 중립  t3: 기쁨   │      │ 핵심 프레임(t3, t7)만 레이블   │
│ t4: 기쁨  t5: 중립  t6: 슬픔   │ ──▶ │  ↓                           │
│ t7: 슬픔  t8: 분노  ...        │      │ 표현학습으로 전체 시퀀스      │
│ 레이블 비용 O(N), 수천 프레임   │      │ 감정 흐름 복원 O(k), k<<N    │
└──────────────────────────────┘      └──────────────────────────────┘
```

### 데이터 효율 학습의 4대 접근

- **Self-Supervised Learning(자기지도학습)**: 레이블 없이 데이터 자체의 구조(대조학습, MAE)로 표현 학습
- **Few-Shot/Zero-Shot**: 소수 예시 또는 자연어 설명만으로 새로운 태스크 일반화
- **Active Learning**: 모델이 레이블링 가치가 높은 샘플을 선택해 사람에게 질의
- **Meta-Learning(학습 방법 학습)**: 다양한 태스크에서 빠르게 적응하는 초기화·알고리즘 학습

### 멀티모달 감정 인식의 입력 모달리티

- **음성(Speech)**: 음향 특징(피치·에너지·스펙트럼), 운율(prosody) 기반
- **얼굴(Face)**: Action Unit, 표정 랜드마크, 미세 표정
- **텍스트(Text)**: 발화 내용의 감정 어휘·극성(polarity), 맥락
- **생체신호(Bio)**: EEG·ECG·GSR — 연구용, 웨어러블 확산과 결합

### 학습·배포 파이프라인

```
원천 데이터(음성·영상) → 핵심 단서 자동 선별(Attention/Saliency)
   ↓
소수 구간 레이블링 + Self-Supervised 사전학습
   ↓
감정 시퀀스 복원 모델 학습(Transformer/Seq2Seq)
   ↓
배포(온디바이스/서버) → Drift 모니터링 → 지속학습(Continual Learning)
```

## 3. 기술사 출제 포인트

### 논술형 예상 주제

- 데이터 레이블링 비용 절감을 위한 AI 학습 방법론(자기지도·준지도·능동학습) 비교
- 멀티모달 감정 인식 시스템 설계 시 모달리티 융합 전략(Early/Late/Hybrid Fusion)
- 지속학습(Continual Learning)과 치명적 망각(Catastrophic Forgetting) 대응 전략

### 단답형 핵심 키워드

- **자기지도학습(Self-Supervised Learning)**: 레이블 없이 데이터 자체의 예측 과제로 표현을 학습하는 패러다임
- **대조학습(Contrastive Learning)**: 유사 쌍은 가깝게, 비유사 쌍은 멀게 표현 공간을 학습
- **표현학습(Representation Learning)**: 태스크에 유용한 잠재 특징을 학습으로 얻는 기법
- **멀티모달 퓨전**: 서로 다른 모달 입력을 통합해 최종 예측에 활용(Early/Late/Hybrid)
- **액티브 러닝(Active Learning)**: 모델 불확실성 높은 샘플을 우선 레이블링해 효율을 극대화
- **Attention/Saliency**: 입력 중 중요 부분을 가중치로 강조하는 기법, 핵심 단서 탐지에 활용

### 비교·대조 포인트

아래는 지도학습 대비 데이터 효율 학습 기법의 차이점 비교다.

| 구분 | 지도학습 | 자기지도학습 | Few-Shot | Active Learning |
|------|---------|-------------|----------|----------------|
| 레이블 요구 | 대량 필요 | 불필요(사전학습) | 소수(수~수십) | 선택적 질의 |
| 성능 기준 | 태스크 특화 | 전이 가능 표현 | 빠른 적응 | 점진 향상 |
| 대표 기법 | CNN/RNN 분류 | SimCLR, BERT, MAE | MAML, Prototypical | Uncertainty Sampling |
| 적용 단계 | 최종 파인튜닝 | 사전학습 | 신규 태스크 | 데이터 구축 단계 |
