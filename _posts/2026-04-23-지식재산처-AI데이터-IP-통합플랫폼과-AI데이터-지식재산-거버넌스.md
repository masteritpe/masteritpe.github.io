## 1. 뉴스 요약

특허뉴스·뉴시스·이데일리·비즈월드·금강일보 등 다수 매체가 보도한 바에 따르면 지식재산처가 'AI·데이터와 지식재산' 통합 웹페이지를 개설해 AI·데이터 분야 지식재산(IP) 이슈·정책·판례·특허 정보를 한곳에서 제공한다. 배경에는 생성형 AI 학습 데이터의 저작권 침해 소송, AI 산출물의 발명자성(Inventorship) 인정, 데이터 자체의 자산화·거래 필요성 등 지식재산 패러다임 변화가 있다. 정부는 2023년 특허청을 '지식재산처'로 승격한 이후 AI·데이터를 포함한 지식재산 5법 제정, 국가 IP 전략 체계 재설계를 추진 중이며, 본 플랫폼은 공공·민간·학계가 일관된 IP 정보·정책에 접근해 AI 시대의 지식재산 분쟁 리스크를 선제적으로 관리하는 거버넌스 허브 역할을 목표로 한다.

## 2. 핵심 개념

### AI·데이터 지식재산 4대 쟁점

```
┌──────────────────────────────────────────────────────────────┐
│ (1) 학습 데이터 저작권                                         │
│   - TDM(Text & Data Mining) 면책 범위                         │
│   - 공정이용(Fair Use) vs 라이선스 필수                       │
├──────────────────────────────────────────────────────────────┤
│ (2) AI 산출물의 권리 귀속                                      │
│   - 발명자·저작자: 자연인만 인정(한국·미국·EU)                │
│   - AI 도구 사용자·기여자·서비스 제공자 간 계약 설계          │
├──────────────────────────────────────────────────────────────┤
│ (3) 데이터 자산 보호                                          │
│   - 데이터 거래·라이선스, 부정경쟁방지법 영업비밀             │
│   - 한국형 데이터 산업법, EU Data Act                         │
├──────────────────────────────────────────────────────────────┤
│ (4) 특허·상표·디자인 AI 활용                                  │
│   - AI 기반 선행기술 검색·유사 상표 탐지                      │
│   - AI 생성 디자인 등록 가능성                                │
└──────────────────────────────────────────────────────────────┘
```

### 지식재산 통합 플랫폼 구조(개념도)

```
       [사용자: 기업·연구자·변리사·정책입안자]
                      │
                      ▼
        ┌─────────────────────────────┐
        │  AI·데이터 IP 통합 웹페이지  │
        │  (지식재산처 단일 창구)     │
        └─────────────────────────────┘
         │          │          │           │
         ▼          ▼          ▼           ▼
   [정책·법령]  [판례·이슈] [특허 DB]   [데이터 IP]
   (IP5법)     (국내·해외)  (KIPRIS 연계) (거래·라이선스)
         │          │          │           │
         └──────────┴──────────┴───────────┘
                      │
                 (국제 WIPO·EUIPO·USPTO 연계)
```

### AI 거버넌스와 지식재산의 접점

- **데이터셋 실사(Data Due Diligence)**: 학습 데이터 출처·라이선스·개인정보 동의 검증
- **AI BOM·Model Card**: 모델 구성요소 투명성, IP 출처 공개
- **출처 표기(Provenance)**: C2PA·콘텐츠 인증(Content Credentials) 표준
- **라이선스 관리**: Creative Commons, OpenRAIL-M, LlamaLicense 등 모델·데이터 라이선스 추적
- **TDM 예외 규정**: EU DSM Directive Art.3·4, 일본 저작권법 30조의4 등

### 국내외 주요 제도·표준

아래는 AI·데이터 지식재산 주요 제도 체계이다.

| 영역 | 국내 | 해외 |
|------|------|------|
| 저작권 | 저작권법 개정 논의, TDM 예외 신설 추진 | EU CDSM Directive, 일본 저작권법 30조의4 |
| 특허 | 특허법상 발명자=자연인, AI 도구화 | USPTO·EPO '자연인만 발명자' 판결 |
| 데이터 | 데이터산업법, 부정경쟁방지법 제2조 '데이터 부정사용' | EU Data Act, Data Governance Act |
| 표준 | KS X 6500 시리즈 데이터 품질 | ISO/IEC 5259(AI 데이터 품질), WIPO 가이드 |
| 기관 | 지식재산처, KIPRIS, KISTA | WIPO, USPTO, EPO, EUIPO |

## 3. 기술사 출제 포인트

### 논술형 예상 주제

- 생성형 AI 학습 데이터의 저작권 침해 논란 대응을 위한 국가 지식재산 거버넌스 체계 설계
- AI 산출물의 권리 귀속·발명자성 쟁점과 기업의 AI 계약·IP 관리 프로세스 구축 방안
- 데이터 자산 시대의 IP 보호(부정경쟁방지·영업비밀·데이터산업법) 통합 전략과 국제 정합성 확보

### 단답형 핵심 키워드

- **TDM(Text & Data Mining) 면책**: AI 학습 목적의 대규모 텍스트·데이터 분석에 대한 저작권 예외
- **AI Inventorship**: AI가 발명자가 될 수 있는가, 현재 한·미·일·EU 모두 자연인 한정
- **C2PA(Coalition for Content Provenance and Authenticity)**: 콘텐츠 출처·편집 이력 서명 표준
- **OpenRAIL-M**: 책임 있는 AI 모델 라이선스, 사용 제한 조항 포함
- **데이터산업법**: 한국 데이터 기본법, 데이터 자산 가치평가·거래 지원
- **지식재산 5법**: 특허·실용신안·디자인·상표·영업비밀(부정경쟁방지법)
- **KIPRIS**: 한국특허정보원 운영 특허정보 검색 서비스
- **EU AI Act + Data Act**: AI 규제와 데이터 접근·공유 의무를 병행 규율

### 비교·대조 포인트

아래는 전통 IP 관리와 AI·데이터 시대 IP 관리의 차이점이다.

| 구분 | 전통 IP 관리 | AI·데이터 IP 관리 |
|------|---------------|---------------------|
| 보호 대상 | 특허·상표·저작물 | + 학습 데이터·모델·AI 산출물 |
| 권리자 | 발명자·저작자(자연인) | 계약으로 설계(플랫폼·제공자·사용자) |
| 출처 증명 | 등록·공증 | + C2PA·전자서명·블록체인 출처 |
| 분쟁 유형 | 침해·무효 소송 | + 데이터 크롤링·TDM·딥페이크 분쟁 |
| 평가 방법 | 기술가치평가(TCV) | + 데이터 자산 평가, 모델 자산 평가 |
| 관리 주체 | IP팀·법무팀 | + AI 거버넌스·DPO·CISO 협업 |
