## 1. 뉴스 요약

행정안전부가 범정부 '공공AI사업지원센터'를 공식 개소하고, 산·학·연 최고 전문가 101명으로 구성된 '인공지능정부 기술자문단'을 출범시켰다. 이와 함께 정부는 에이전틱 AI 지원을 위한 민관 협력 4대 분과를 가동하며 'AI 민주정부' 전환을 본격화하고 있다. 에너지공단은 'AI 감사·윤리 헌장'을 선포하여 공공기관의 무분별한 AI 도입을 모니터링하겠다고 밝혔다. 이는 공공부문 AI 전환(AX)이 단순 기술 도입을 넘어 거버넌스·윤리·조직 혁신을 포괄하는 종합 전략으로 발전하고 있음을 보여준다.

## 2. 핵심 개념

### 공공 AI 추진 체계 구조

```
┌──────────────────────────────────────────────────────┐
│              공공 AI 추진 거버넌스 체계                 │
├──────────────────────────────────────────────────────┤
│                                                      │
│  ┌──────────────────────────────────────────────┐    │
│  │         정책 계층 (Policy Layer)              │    │
│  │  - 국가AI위원회 / 과기정통부 / 행안부          │    │
│  │  - AI 기본법, AI 윤리 기준                    │    │
│  └──────────────┬───────────────────────────────┘    │
│                 ↓                                    │
│  ┌──────────────────────────────────────────────┐    │
│  │       지원 계층 (Support Layer)               │    │
│  │  - 공공AI사업지원센터 (컨설팅·기술 지원)       │    │
│  │  - AI정부 기술자문단 (산학연 101명)            │    │
│  │  - AI혁신추진단 (중소기업 AX 전담)             │    │
│  └──────────────┬───────────────────────────────┘    │
│                 ↓                                    │
│  ┌──────────────────────────────────────────────┐    │
│  │       실행 계층 (Execution Layer)             │    │
│  │  - 각 부처/공공기관 AI 사업 추진               │    │
│  │  - AI 감사·윤리 헌장 준수                      │    │
│  │  - 규제프리 AI 실증특구 운영                   │    │
│  └──────────────────────────────────────────────┘    │
└──────────────────────────────────────────────────────┘
```

### 공공 AI 도입 성숙도 모델

| 단계 | 수준 | 특징 | 핵심 활동 |
|------|------|------|----------|
| Level 1 | 인식(Awareness) | AI 필요성 인지 | 교육, 파일럿 기획 |
| Level 2 | 실험(Pilot) | 개별 부서 PoC 수행 | 소규모 실증, 데이터 정비 |
| Level 3 | 운영(Operational) | 특정 업무에 AI 적용 | 시스템 통합, 성과 측정 |
| Level 4 | 확산(Scaling) | 조직 전반 AI 확산 | 플랫폼화, 거버넌스 체계 |
| Level 5 | 최적화(Optimized) | AI 기반 의사결정 내재화 | 자동화, 지속 개선, AI 민주정부 |

## 3. 기술사 출제 포인트

### 예상 문제
- **공공부문 AI 전환(AX) 추진 전략과 거버넌스 체계를 설명하고, 성공적 도입을 위한 핵심 요소를 논하시오**
- **공공 AI 서비스의 윤리적 도입을 위한 프레임워크를 설계하고, AI 영향평가 절차를 설명하시오**
- **공공데이터 기반 AI 서비스 구축 시 데이터 품질 관리와 개인정보보호 방안을 논하시오**

### 답안 작성 키워드
- 공공 AI 거버넌스: 정책-지원-실행 3계층 체계
- AI 성숙도 모델: 인식→실험→운영→확산→최적화
- AI 영향평가(AIA): 위험도 기반 사전 평가 제도
- AI 윤리 6원칙: 투명성, 공정성, 책임성, 안전성, 개인정보보호, 포용성
- 데이터 기반 행정: 공공데이터법, 데이터 품질관리

### 공공 AI vs 민간 AI 도입 비교

| 구분 | 공공 AI | 민간 AI |
|------|--------|--------|
| 목표 | 공공 가치·국민 편의 | 수익·효율 극대화 |
| 규제 수준 | 높음(법적 근거 필수) | 상대적 유연 |
| 데이터 특성 | 개인정보 다수, 공공데이터 | 고객·거래 데이터 |
| 의사결정 투명성 | 필수(행정절차법) | 권장 수준 |
| 위험 관리 | AI 영향평가 의무화 추세 | 자율 관리 |
| 조달 방식 | 공공조달(나라장터) | 자유 계약 |
| 성과 지표 | 국민 만족도, 행정 효율 | ROI, 매출 기여 |

## 4. 관련 토픽 연계

| 연계 토픽 | 연관성 |
|----------|--------|
| AI 거버넌스 | 공공 AI 윤리·규제 프레임워크의 상위 체계 |
| 디지털 전환(DX/AX) | 공공부문 디지털 전환의 AI 중심 진화 |
| 클라우드 컴퓨팅(CSAP) | 공공 AI 인프라의 클라우드 전환 기반 |
| 공공데이터 | AI 학습용 공공데이터 개방·품질 관리 |
| 개인정보보호(PIPA) | 공공 AI의 개인정보 영향평가(PIA) 연계 |
| IT 서비스 관리(ITSM) | AI 기반 IT 자산 관리·운영 혁신 |
| 설명가능한 AI(XAI) | 공공 AI 투명성·책임성 확보 핵심 기술 |
