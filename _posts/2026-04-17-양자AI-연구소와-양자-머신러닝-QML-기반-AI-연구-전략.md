## 1. 뉴스 요약

국립부경대학교가 국내 대학 최초로 '양자AI연구소'를 공식 개소했다. 양자 컴퓨팅과 AI를 결합한 양자 머신러닝(QML) 전문 연구기관으로, 양자 알고리즘 설계·양자 신호처리·양자 기반 최적화를 핵심 연구영역으로 설정했다. 부산일보·네이트 등 다수 매체가 '국내 대학 최초'라는 상징성에 주목해 보도했다. 같은 기간 엔비디아는 양자 컴퓨팅 연계 AI 신기술을 발표했고, 정부도 양자·AI 융합 R&D 확대 기조를 유지하고 있어 양자 AI가 기술사 시험에서도 본격적인 주제로 부상할 전망이다.

## 2. 핵심 개념

### 양자 머신러닝(QML) 전체 구조

```
┌──────────────────────────────────────────────────────────────┐
│         양자 머신러닝(QML) 하이브리드 연산 아키텍처            │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  [고전 컴퓨터 계층]                                            │
│   데이터 전처리 · 파라미터 최적화(Adam 등)                      │
│                  │                                           │
│                  ▼  데이터 인코딩                              │
│  [양자 인코딩 계층]                                            │
│   Amplitude Encoding · Angle Encoding · Basis Encoding        │
│                  │                                           │
│                  ▼                                           │
│  [양자 연산 계층 (QPU)]                                       │
│   변분양자회로(VQC) · 양자 커널 · 양자 상태 중첩/얽힘           │
│   Ansatz: Hardware-efficient / QAOA / VQE                     │
│                  │                                           │
│                  ▼  측정(Measurement)                         │
│  [고전 후처리 계층]                                            │
│   기대값 → 손실함수 계산 → 파라미터 업데이트(역전파 아님)        │
│                  │                                           │
│                  └──→ 반복(Hybrid Loop)                       │
│                                                              │
│  ※ 현재 NISQ 시대에서 고전+양자 하이브리드가 실용 해법          │
└──────────────────────────────────────────────────────────────┘
```

### 주요 QML 알고리즘과 특징

| 알고리즘 | 용도 | 이론적 이점 | 현실 제약 |
|---------|-----|-----------|----------|
| HHL (선형 시스템) | 행렬 방정식 해법 | 지수적 가속(O(log N)) | 큐빗 수·QRAM 필요 |
| VQE (변분 양자) | 기저상태 에너지/분자 | NISQ에서 실행 가능 | Barren Plateau |
| QAOA (조합최적화) | MaxCut, 포트폴리오 | 근사 최적해 가속 | 회로 깊이 한계 |
| QSVM/Quantum Kernel | 분류·회귀 | 고차원 특성공간 | 데이터 인코딩 비용 |
| QGAN | 데이터 생성 | 생성 샘플링 가속 | 학습 불안정 |
| Quantum Transformer | 언어/패턴 인식 | 주목 메커니즘 표현력↑ | 큐빗·디코히런스 |

### 고전 AI vs 양자 AI(QML) 비교

| 구분 | 고전 머신러닝 | 양자 머신러닝(QML) |
|------|-------------|------------------|
| 연산 단위 | 비트(0/1) | 큐빗(중첩·얽힘) |
| 특성공간 | 유한 차원 고전공간 | 힐베르트 공간(지수 차원) |
| 병렬성 | 하드웨어 병렬 | 양자 중첩 기반 내재 병렬 |
| 학습 방식 | 역전파(미분) | 파라미터 시프트 규칙 |
| 주요 병목 | 메모리·GPU | 디코히런스·큐빗 수 |
| 양자 어드밴티지 | - | 특정 문제(HHL 등)에서 지수적 가속 |
| 성숙도 | 산업 성숙 | NISQ(초기)~FTQC(미래) |

## 3. 기술사 출제 포인트

### 예상 문제
1. **양자 머신러닝(QML)의 개념, 주요 알고리즘, 고전 머신러닝과의 차이를 설명하시오**
   - 큐빗·중첩·얽힘의 정보이론적 의미와 힐베르트 공간 표현
   - VQE/QAOA/HHL/Quantum Kernel 등 주요 알고리즘 용도별 정리
   - 파라미터 시프트 규칙 기반 학습과 Barren Plateau 문제
   - NISQ 시대 고전-양자 하이브리드 접근의 현실성

2. **양자 AI 활용 방안과 기술적 제약을 논하시오**
   - 응용: 신약개발(분자 시뮬레이션), 금융 최적화, 물류 경로
   - 제약: 디코히런스, 노이즈, 큐빗 수, QRAM 부재
   - 양자 오류정정(QEC)·논리 큐빗의 역할
   - 고전 AI(트랜스포머)와 양자 AI의 상호보완적 결합 전략

3. **양자컴퓨팅 시대의 AI 데이터분석 전략을 제시하시오**
   - 데이터 인코딩 전략(진폭/각도/기저) 선택 기준
   - 하이브리드 파이프라인 설계: 고전 전처리 + 양자 커널 + 고전 후처리
   - 양자 시뮬레이터(PennyLane/Qiskit) 기반 점진적 도입 로드맵
   - 양자 친화 문제의 식별 기준(Quantum Utility)
