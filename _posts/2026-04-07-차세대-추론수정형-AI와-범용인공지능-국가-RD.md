## 뉴스 요약

2026년 4월 6일, 과학기술정보통신부의 **차세대 인공지능 기술개발 사업** 2단계 과제 주관기관으로 **경북대학교 정희철 교수팀**이 선정되었다. 전자신문·뉴시스·다음·경북신문 등 7개 매체가 동시 보도했으며, 핵심 키워드는 **"스스로 추론하고 수정하는 AI"**, 즉 **자기수정(Self-Correction) 기반 범용 인공지능(AGI)** 개발이다. 1단계가 파운데이션 모델 학습 인프라 구축에 집중했다면, 2단계는 **추론(Reasoning)·검증(Verification)·자기수정 루프**를 통한 신뢰성 강화에 무게를 둔다.

## 핵심 개념

차세대 추론형 AI는 단순 다음-토큰 예측을 넘어, 모델이 **사고 단계를 명시적으로 생성·검증·수정**하는 구조를 갖는다.

- **Chain-of-Thought / Tree-of-Thought**: 중간 추론 단계 명시화로 복잡 문제 분해.
- **Self-Consistency**: 다중 추론 경로 샘플링 후 다수결로 답 안정화.
- **Self-Refine / Self-Correction**: 모델이 자기 출력을 비평(Critic)하고 재작성.
- **Process Reward Model(PRM)**: 결과(Outcome)가 아닌 추론 과정 단계마다 보상을 학습.
- **Verifier-Guided Decoding**: 수학·코딩 등에서 외부 검증기로 후보를 가지치기.
- **Test-Time Compute Scaling**: 추론 시점에 연산을 늘려 정확도를 끌어올리는 새 스케일링 축.

이는 OpenAI o-시리즈, DeepSeek-R1 등에서 검증된 경향이며, 한국 국가 R&D가 이 방향으로 본격 진입한다는 신호다.

## 차세대 AI vs 기존 LLM 비교

다음은 차세대 추론형 AI와 기존 LLM의 주요 차이를 정리한 것이다.

| 구분 | 기존 LLM | 추론·수정형 AI |
|---|---|---|
| 학습 목표 | 다음 토큰 예측 | 추론 경로 + 결과 보상 |
| 보상 모델 | ORM(결과 보상) | PRM(과정 보상) |
| 추론 방식 | 단일 패스 | 다단계·자기수정 루프 |
| 스케일링 축 | 모델·데이터 크기 | + 추론 시점 연산량 |
| 주요 약점 | 환각, 산술·논리 취약 | 비용·지연 증가 |
| 평가 지표 | MMLU, HELM | GPQA, AIME, SWE-Bench |

## 기술사 출제 포인트

- **AGI vs 범용 AI 정의 차이**, 단계별 발전 모델(L1~L5).
- **RLHF → RLAIF → PRM 기반 RL** 진화 흐름과 보상 해킹(Reward Hacking) 문제.
- **추론 시점 스케일링(Inference-Time Scaling)**의 비용·지연·전력 트레이드오프.
- **국가 R&D 거버넌스**: 단계평가(Stage-Gate), 컨소시엄, 1·2단계 연계 평가 체계.
- **신뢰성·설명가능성(XAI)**과 추론 과정 가시화의 결합.
- **데이터 품질**: 고품질 추론 학습 데이터(STEM, 코드, 검증 가능한 문제) 확보 전략.
