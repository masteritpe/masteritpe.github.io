## 1. 뉴스 요약

2026년 3월 셋째 주, 제로트러스트(Zero Trust) 보안 관련 다수의 뉴스가 쏟아졌다. 엑소스피어랩스는 WIS 2026에서 제로트러스트 보안 플랜을 공개했고, 파이오링크는 네트워크 통신 시작점부터 제로트러스트를 적용하는 '티프론트 ZT'를 선보였다. 세이퍼존은 오프라인 엔드포인트까지 커버하는 제로트러스트 솔루션을 출시했으며, 옥타코는 IAM/SSO 기업 넷츠와 Phishing-Resistant 기반 제로트러스트 시장 공략을 위한 MOU를 체결했다. 한편 SKT-에릭슨은 AI-RAN과 제로트러스트를 결합한 6G 핵심 네트워크 기술 공동 개발에 나섰다.

## 2. 핵심 개념

### 제로트러스트 아키텍처(ZTA)
- **정의**: "Never Trust, Always Verify" 원칙에 기반한 보안 모델. 내/외부 네트워크를 구분하지 않고 모든 접근 요청에 대해 지속적으로 검증
- **NIST SP 800-207**: 제로트러스트 아키텍처의 표준 프레임워크
  - PEP(Policy Enforcement Point): 정책 집행 지점
  - PDP(Policy Decision Point): 정책 결정 지점 (PE + PA)
  - PE(Policy Engine): 접근 허용/거부 판단
  - PA(Policy Administrator): 세션 설정/해제 관리

### 제로트러스트 핵심 구성요소

| 구성요소 | 설명 |
|----------|------|
| 사용자 인증 | MFA, FIDO2, Passwordless 인증 |
| 디바이스 보안 | 엔드포인트 상태 검증, MDM 연동 |
| 네트워크 세분화 | 마이크로세그멘테이션, SDP |
| 애플리케이션 접근 | ZTNA(Zero Trust Network Access) |
| 데이터 보호 | DLP, 암호화, 최소 권한 원칙 |
| 가시성/분석 | SIEM, SOAR, XDR 연동 |

### 엔드포인트 제로트러스트
세이퍼존이 선보인 **오프라인 엔드포인트 제로트러스트**는 네트워크 연결 없이도 디바이스 신뢰를 검증하는 접근 방식으로, 폐쇄망 환경의 보안을 강화한다.

### Phishing-Resistant 인증
옥타코-넷츠 협력의 핵심인 **피싱 저항 인증**은 FIDO2/WebAuthn 기반으로 자격증명 탈취에 의한 우회를 원천 차단한다.

## 3. 기술사 출제 포인트

> **예상 문제**: 제로트러스트 아키텍처(ZTA)의 개념과 NIST SP 800-207 기반 구성요소를 설명하고, 기존 경계 기반 보안 모델과의 차이점을 비교하시오.

- **경계 기반 보안 vs 제로트러스트**: 성벽(Castle-and-Moat) 모델의 한계 → 내부 위협, 클라우드 확산, 원격근무
- **SDP(Software Defined Perimeter)**: 제로트러스트 구현 기술, 컨트롤러 기반 접근제어
- **마이크로세그멘테이션**: 네트워크를 세분화하여 횡적 이동(Lateral Movement) 차단
- **지속적 인증(Continuous Authentication)**: 세션 유지 중에도 컨텍스트 기반 재검증
- **6G 네트워크와 제로트러스트**: SKT-에릭슨 사례처럼 차세대 통신망에 보안 내재화(Security by Design)

## 4. 관련 토픽 연계

- **네트워크**: SDP, SASE(Secure Access Service Edge), SD-WAN
- **시스템구조**: 클라우드 네이티브 보안, 컨테이너 보안
- **IT경영**: 정보보호 거버넌스, ISMS-P, ISO 27001
- **AI데이터분석**: AI 기반 이상행위 탐지(UEBA), SOAR 자동화
