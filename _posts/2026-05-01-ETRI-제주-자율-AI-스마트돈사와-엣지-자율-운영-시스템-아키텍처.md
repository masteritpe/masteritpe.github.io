## 1. 뉴스 요약

인공지능신문·동아사이언스·IT비즈뉴스 등 보도에 따르면, **한국전자통신연구원(ETRI)이 제주도 양돈농가에 「AI 스스로 판단하는 자율 스마트돈사(AX 스마트팜)」를 국내 처음으로 구축**했다. 핵심은 ① **돈사 내 환기·온습도·사료·음수·환경위생을 사람의 개입 없이 AI가 24시간 의사결정·제어**하고, ② **돼지 행동·생체 데이터를 영상·음향·센서로 실시간 수집**해 질병·발정·스트레스를 조기 탐지하며, ③ **돈사 단(엣지)에서 추론·제어를 수행하고 클라우드는 학습·정책 갱신**을 담당한다는 점이다. 동시에 **「생산성 향상과 탄소중립을 동시 달성」**하는 정량 지표(메탄 배출 저감·사료 효율)를 KPI로 설정했다. 배경에는 **농업 인력 고령화·축산 메탄 규제·아프리카돼지열병(ASF) 등 가축전염병 리스크**가 있다. 이번 사례는 **공장·발전소 등 산업 자율 운영(Autonomous Operations)이 농축산까지 확장**되는 흐름을 보여주며, 시스템구조 관점에서는 **엣지 추론 + 클라우드 학습 + 디지털트윈 + 폐쇄망 보안 + 안전 페일세이프**가 결합한 **사이버-피지컬 시스템(CPS)** 설계 과제를 던진다.

## 2. 핵심 개념

### AX 스마트돈사 자율 운영 시스템 계층 구조

```
┌──────────────────────────────────────────────────────────┐
│ ⑤ 클라우드 : 모델 학습·정책·디지털트윈·통계               │
├──────────────────────────────────────────────────────────┤
│ ④ 엣지 게이트웨이 : 경량 LLM/CV 추론·로컬 의사결정         │
├──────────────────────────────────────────────────────────┤
│ ③ 제어 : PLC·VFD·환기·급이·조명 액추에이터                │
├──────────────────────────────────────────────────────────┤
│ ② 데이터 수집 : 카메라·마이크·온습도·암모니아·체중계         │
├──────────────────────────────────────────────────────────┤
│ ① 물리 환경 : 돈방·돼지·사료·음수·환기 시스템              │
└──────────────────────────────────────────────────────────┘
```

### 기존 스마트팜 vs 자율 스마트돈사

다음 표는 두 모델의 차이를 정리한다.

| 구분 | 기존 스마트팜 | 자율 AX 스마트돈사 |
|------|---------------|---------------------|
| 의사결정 | 사람·룰 기반 자동제어 | AI 자율 판단·제어 |
| 데이터 | 온습도·CO2 등 환경 위주 | 영상·음향·생체·환경 융합 |
| 추론 위치 | 클라우드 중심 | 엣지(돈사) 중심 + 클라우드 학습 |
| 이상 탐지 | 임계치 알람 | 행동·울음·체온 다중 모달 이상 탐지 |
| 페일세이프 | 수동 복구 | 안전 모드·이중화·인간 호출 |
| KPI | 사료비·환경 안정성 | 사료 효율·메탄·질병 조기 탐지율 |

### 엣지 자율 운영 핵심 구성요소

- **다중 모달 인식**: 영상(Vision Transformer)·음향(MFCC+CNN)·생체(IoT)
- **자율 의사결정**: RL 기반 환기·급이 정책, 안전 제약 만족(Constrained RL)
- **디지털트윈**: 돈사 3D 모델·기류·열·축군 시뮬레이션, 정책 시험장
- **MLOps**: 농가 분산 학습(Federated Learning), 모델 버전·롤백
- **사이버-피지컬 안전**: 페일세이프 룰, 안전 인증 PLC, 비상 수동전환
- **네트워크**: 농촌 LTE·LoRa·로컬 Wi-Fi 6, 통신 단절 시 엣지 단독 운영

### 도입 시 고려사항

- **데이터 라벨링·바이오시큐리티**: 농가 데이터 수집은 방역(외부 출입 제한)과 충돌
- **모델 일반화**: 농가별 환경·품종 차이 → 농가 적응형 미세조정(Per-Farm Fine-tune)
- **엣지 자원 한계**: GPU 없이도 동작하는 INT8 양자화·증류 모델 필요
- **안전·법규**: 동물 복지·축산법, 자동제어 사고 시 책임 소재(농가·통합사·SI)
- **지속가능성**: 메탄 측정·탄소 배출권(MRV), RE100·태양광 결합
- **확장성**: 양돈→가금→낙농→시설원예로의 플랫폼 확장 설계

## 3. 기술사 출제 포인트

- **자율 운영(Autonomous Operations) 시스템의 정의·필요성**: 인력·환경 압력
- **기존 스마트팜과 자율 AX 스마트팜 비교**: 의사결정·데이터·KPI 차이
- **사이버-피지컬 시스템(CPS) 5계층 아키텍처와 농축산 적용**
- **엣지 AI vs 클라우드 AI의 역할 분담**: 추론·학습·정책 갱신
- **다중 모달(영상·음향·생체) 이상 탐지 모델 설계**
- **연합학습(Federated Learning) 기반 농가 분산 학습 구조**
- **디지털트윈을 활용한 정책 시뮬레이션과 안전 검증**
- **자율 시스템의 페일세이프·안전 인증·인간 개입(HITL) 설계**
- **농축산 IoT 통신 환경(LoRa·LTE-M·Wi-Fi 6)과 단절 대비 운영**
- **탄소중립·MRV 연계와 자율 운영 KPI 설계**
